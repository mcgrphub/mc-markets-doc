This reference covers public MC OpenAPI endpoints. The sidebar groups **Market data**, **Trading**, and **Finance**.

All endpoints share the same gateway host, authentication scheme (AK/SK with HMAC-SHA256), and JSON response envelope. Request paths still include the service segment used for signing (for example `/openapi/v1/mc-account/...`).

## Endpoint groups

| Group | Includes |
| --- | --- |
| Market data | Market data, Currency, Exchange rate |
| Trading | Orders and positions, Trading account, Orders, Deals, Account trading records, Deal orders, Funding fee history, Trading symbols, Funding rate |
| Finance | Finance account, Vault account, Fund account |

Use the sidebar to open each group. The interactive playground is disabled because requests require custom AK/SK signing — follow [Authentication](/authentication) and [First request](/guides/first-request).
