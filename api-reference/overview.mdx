## API at a glance

The table below summarises the key properties of the Open API Platform REST API.

| Property    | Value                                          |
| ----------- | ---------------------------------------------- |
| Base URL    | `https://api.example.com/v1`                   |
| Protocol    | HTTPS only                                     |
| Format      | JSON                                           |
| Auth        | Bearer token (API Key)                         |
| Versioning  | URL path (`/v1`)                               |
| Rate limits | See [/guides/rate-limits](/guides/rate-limits) |
