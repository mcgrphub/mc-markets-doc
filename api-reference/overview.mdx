This reference covers public MC OpenAPI endpoints (the `@OpenApiPermission`-filtered subset in the Chinese Specs). Sidebar groups **Account**, **Trade**, **Risk**, and **Aggregator** are hand-written MDX pages, one page per external tag.

All endpoints share the same gateway host, authentication scheme (AK/SK with HMAC-SHA256), and JSON response envelope. Request paths already include the service segment used for signing (for example `/openapi/v1/mc-account/...`).

## Endpoint groups

| Group | Spec | Gateway path prefix |
| --- | --- | --- |
| Account | `openapi/mc-account.json` | `/openapi/v1/mc-account/` |
| Trade | `openapi/mc-trade.json` | `/openapi/v1/mc-trade/` |
| Risk | `openapi/mc-risk.json` | `/openapi/v1/mc-risk/` |
| Aggregator | `openapi/mc-aggregator.json` | `/openapi/v1/mc-aggregator/` |

Use the sidebar to open each group. The interactive playground is disabled because requests require custom AK/SK signing — follow [Authentication](/authentication) and [First request](/guides/first-request).
