## 理财账户提现（活期）

`POST /openapi/v1/mc-account/finance/account/withdraw` — permission `TRADE`.

仅支持活期产品：传入产品代码与提现金额，从理财转出到资金账户再划入现货。定期多笔持仓请使用「定期持仓提现」接口并指定持仓明细 id

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| trackingId | string | Yes | 用于跟踪请求的唯一标识 |
| productCode | string | Yes | 活期理财产品代码 |
| amount | number | Yes | 提现金额 |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/finance/account/withdraw" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```

## 理财定期持仓提现

`POST /openapi/v1/mc-account/finance/account/withdraw/fixed` — permission `TRADE`.

仅支持定期产品：指定持仓明细 id（持有中或已到期均可；与当前持仓接口 positionId 一致），后端按该笔本金+利息全额赎回，工作流与活期提现一致

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| trackingId | string | Yes | 用于跟踪请求的唯一标识 |
| positionId | integer | Yes | 定期持仓主键 id（持有中或已到期均可赎回；与当前持仓接口返回的 positionId 一致） |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/finance/account/withdraw/fixed" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```

## 理财产品目录

`GET /openapi/v1/mc-account/finance/products/catalog` — permission `READ`.

按类型（活期/定期）查询已启用产品配置全量列表（不分页），按基础年化+超额奖励年化降序；查询参数仅 type 必填

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| type | string | Yes | 产品类型：活期 / 定期 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/finance/products/catalog" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "type=VALUE"
```

## 理财概览

`GET /openapi/v1/mc-account/finance/account/overview` — permission `READ`.

按产品维度返回 products，顺序为 product_code 升序、再 currency 升序；用户侧 userDepositedAmount/userInterestAmount/availableBalance 来自该产品在 ac_user_finance_position 的持仓汇总（该产品币种）。totalDepositValue 为各产品持仓本金×预言机价折 USDC 之和。totalInterestValue、yesterdayInterestValue 仍为结息流水按币种折 USDC。userSpotAvailableBalance 为现货币种维度，同币种产品值相同。未登录时用户侧为0

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| totalDepositValue | number | 用户存入总额（各币种本金 × 对应预言机价，折 USDC；未登录为0） |
| totalInterestValue | number | 用户历史结息总额（ac_finance_transaction 成功 INTEREST 按币种汇总 amount 后 × 同概览预言机价，折 USDC；计算口径与 totalDepositValue、yesterdayInterestVal |
| yesterdayInterestValue | number | 昨日利息（自然日昨日 Asia/Shanghai 已入账结息流水，按同套预言机折 USDC；未登录为0） |
| products | array | 按产品维度：每条对应 ac_finance_product + 配置；用户金额来自该产品下持仓汇总，平台字段来自产品汇总表；含 subscribedAmount、fundPoolAmount、availableSubscriptionBal |
| totalDepositValueDisplay | string | 用户存入总额展示值 |
| totalInterestValueDisplay | string | 用户历史结息总额展示值 |
| yesterdayInterestValueDisplay | string | 昨日利息展示值 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/finance/account/overview" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```
