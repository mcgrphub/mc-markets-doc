Paths in this group use the `/openapi/v1/mc-account/` prefix.

## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| GET | /openapi/v1/mc-account/symbol/list | READ | Query symbol network list |
| GET | /openapi/v1/mc-account/symbol/exchange/list | READ | Query exchangeable symbol list |
| GET | /openapi/v1/mc-account/symbol/detail/&#123;id&#125; | READ | Query symbol details |

<Note>
  The interactive Playground is disabled. Sign requests with AK/SK as described in [Authentication](/authentication). See [Base URL](/api-reference/base-url) for the gateway host.
</Note>

## Query symbol network list

`GET /openapi/v1/mc-account/symbol/list` — permission `READ`.

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/symbol/list" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```

## Query symbol details

`GET /openapi/v1/mc-account/symbol/detail/{id}` — permission `READ`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| id | string | Yes | Symbol ID |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| symbolId | integer | Symbol ID |
| assetType | string | Symbol type |
| symbol | string | Symbol name |
| network | string | Symbol network protocol |
| minTradeUnit | number | Minimum movement unit |
| pricePrecision | integer | Price precision |
| exchangeRate | number | Exchange rate |
| minRechargeAmount | number |  |
| minWithdrawAmount | number |  |
| assetNet | string |  |
| iconData | string |  |
| depositStatus | integer |  |
| withdrawalStatus | integer |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/symbol/detail/{id}" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "id=VALUE"
```
