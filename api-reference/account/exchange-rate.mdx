## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| GET | /openapi/v1/mc-account/exchange-rate/query/&#123;symbolType&#125;/&#123;baseCurrency&#125;/&#123;targetCurrency&#125; | READ | Query exchange rate |

## Query exchange rate

`GET /openapi/v1/mc-account/exchange-rate/query/{symbolType}/{baseCurrency}/{targetCurrency}` — permission `READ`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| symbolType | string | Yes | Asset type |
| baseCurrency | string | Yes | Base currency code (ISO 4217) |
| targetCurrency | string | Yes | Target currency code (ISO 4217) |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| data | number | `data` is a scalar of this type |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/{symbolType}/{baseCurrency}/{targetCurrency}" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "symbolType=VALUE" \
  --data-urlencode "baseCurrency=VALUE" \
  --data-urlencode "targetCurrency=VALUE"
```

## Query USD exchange rate

`GET /openapi/v1/mc-account/exchange-rate/query/usd` — permission `READ`.

### Response `data` fields

`data` is an array of objects. Element fields follow.

`data[]` items:

| Field | Type | Description |
| --- | --- | --- |
| symbolType | string | Base currency type |
| baseCurrency | string | Base currency code (ISO 4217 standard) |
| targetCurrency | string | Target currency code (ISO 4217 standard) |
| exchangeRate | number | Exchange Rate |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/usd" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```

## Query flash conversion exchange rate

`GET /openapi/v1/mc-account/exchange-rate/convert-rate/{from}/{to}` — permission `READ`.

For USDT to BTC, set `from` to USDT and `to` to BTC.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| from | string | Yes | Source symbol / symbol to convert from |
| to | string | Yes | Target symbol / symbol to convert to |
| fromAmount | number | No | User-entered amount of the source symbol (`from`); used to estimate fees and expected receipt |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| exchangeRate | number | Exchange Rate |
| timestamp | integer | Timestamp |
| oneFromToQuantity | number | Quantity of [to] currency exchangeable for 1 [from] currency |
| oneToFromQuantity | number | Quantity of [from] currency exchangeable for 1 [to] currency |
| bookPriceDotMap | object | Quotes from the exchange rate conversion process - includes multiple conversion steps, key=currency pair (Warning: only available in non-production environments, used for testing exchange rate accuracy) |
| baseExchangeRate | number | Base exchange rate (before fee deduction / free quota application), used for testing verification |
| baseExchangeRateDisplay | string | Base exchange rate display value |
| fromAmount | number | User-input source currency amount (from amount) |
| fromAmountDisplay | string | User-input source currency amount display value |
| fromAmountScale | integer | User-input source currency amount display precision |
| freeQuotaFromAmount | number | Today's remaining free quota (from amount, used for calculating fee-free tier) |
| freeQuotaFromAmountDisplay | string | Today's remaining free quota (from amount) display value |
| feeAmount | number | Actual fee charged (from amount) |
| feeAmountDisplay | string | Actual fee charged (from amount) display value |
| feeAmountScale | integer | Actual fee charged (from amount) display precision |
| netFromAmount | number | Net amount actually used in exchange (from amount) = fromAmount - feeAmount |
| netFromAmountDisplay | string | Net amount actually used in exchange (from amount) display value |
| toAmount | number | Expected received amount (to amount) |
| toAmountDisplay | string | Expected received amount (to amount) display value |
| toAmountScale | integer | Expected received amount (to amount) display precision |
| convertFeePercent | number | Flash exchange fee percentage (deducted from the source amount) |
| dailyFreeConvertFeeQuotaUsd | number | Daily free flash exchange fee quota (USDC equivalent, accumulated per user, reset at UTC+8 daily cutover) |
| dailyRemainingFreeConvertFeeQuotaUsd | number | Today's remaining free flash exchange fee quota (USDC equivalent, accumulated per user, reset at UTC+8 daily cutover) |
| dailyRemainingFreeConvertFeeQuotaUsdDisplay | string | Today's remaining free flash exchange fee quota (USDC equivalent) display value |
| expireTimestamp | integer | Quote expiry timestamp, in milliseconds |
| quoteSessionId | string | Quote Session ID |

`bookPriceDotMap` fields:

| Field | Type | Description |
| --- | --- | --- |
| <key> | object | Quotes from the exchange rate conversion process - includes multiple conversion steps, key=currency pair (Warning: only available in non-production environments, used for testing exchange rate accuracy) |

`bookPriceDotMap.<key>` map value fields:

| Field | Type | Description |
| --- | --- | --- |
| buyPrice | number | Bid Price |
| sellPrice | number | Ask Price |
| midPrice | number | Mid Price |
| timestamp | integer | Timestamp |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/convert-rate/{from}/{to}" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "from=1755792000" \
  --data-urlencode "to=1755792000"
```
