## Query exchange rate

`GET /openapi/v1/mc-account/exchange-rate/query/{symbolType}/{baseCurrency}/{targetCurrency}` — permission `READ`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| symbolType | string | Yes | Asset type |
| baseCurrency | string | Yes | Base currency code (ISO 4217) |
| targetCurrency | string | Yes | Target currency code (ISO 4217) |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/{symbolType}/{baseCurrency}/{targetCurrency}" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "symbolType=VALUE" \
  --data-urlencode "baseCurrency=VALUE" \
  --data-urlencode "targetCurrency=VALUE"
```

## Query flash conversion exchange rate

`GET /openapi/v1/mc-account/exchange-rate/convert-rate/{from}/{to}` — permission `READ`.

For USDT to BTC, set `from` to USDT and `to` to BTC.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| from | string | Yes | Source symbol / symbol to convert from |
| to | string | Yes | Target symbol / symbol to convert to |
| fromAmount | number | No | User-entered amount of the source symbol (`from`); used to estimate fees and expected receipt |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| exchangeRate | number | Exchange rate |
| timestamp | integer |  |
| oneFromToQuantity | number | Amount of `[to]` per 1 `[from]` |
| oneToFromQuantity | number | Amount of `[from]` per 1 `[to]` |
| bookPriceDotMap | object | Quotes from each step of the exchange-rate conversion; key = currency pair (warning: populated only in non-production, for rate accuracy testing) |
| baseExchangeRate | number | Base exchange rate (before fees / free quota); for testing |
| baseExchangeRateDisplay | string | Base exchange rate display value |
| fromAmount | number | User-entered source amount (`from`) |
| fromAmountDisplay | string | User-entered source amount display value |
| fromAmountScale | integer | User-entered source amount display precision |
| freeQuotaFromAmount | number | Remaining daily free quota (`from` amount; used for fee-free range) |
| freeQuotaFromAmountDisplay | string | Remaining daily free quota (`from` amount) display value |
| feeAmount | number | Actual fee charged (`from` amount) |
| feeAmountDisplay | string | Actual fee display value (`from` amount) |
| feeAmountScale | integer | Actual fee display precision (`from` amount) |
| netFromAmount | number | Net amount participating in conversion (`from`) = fromAmount - feeAmount |
| netFromAmountDisplay | string | Net conversion amount display value (`from`) |
| toAmount | number | Expected received amount (`to`) |
| toAmountDisplay | string | Expected received amount display value (`to`) |
| toAmountScale | integer | Expected received amount display precision (`to`) |
| convertFeePercent | number | Flash conversion fee percentage (deducted from source amount) |
| dailyFreeConvertFeeQuotaUsd | number | Daily free flash conversion fee quota (USDC equivalent; per-user cumulative; UTC+8 day boundary) |
| dailyRemainingFreeConvertFeeQuotaUsd | number | Remaining free flash conversion fee quota today (USDC equivalent; per-user cumulative; UTC+8 day boundary) |
| dailyRemainingFreeConvertFeeQuotaUsdDisplay | string | Remaining free quota display value (USDC equivalent) |
| expireTimestamp | integer | Quote expiration timestamp (ms) |
| quoteSessionId | string | Quote session ID |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/convert-rate/{from}/{to}" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "from=1755792000" \
  --data-urlencode "to=1755792000"
```
