| Method | Path | Permission | Description |

| GET | /openapi/v1/mc-account/mlp/withdraw-info | READ | Vault account withdrawal info |

| GET | /openapi/v1/mc-account/mlp/treasury-info | READ | Vault account treasury info |

  The interactive Playground is disabled. Sign requests with AK/SK as described in [Authentication](/authentication). See [Base URL](/api-reference/base-url) for the gateway host.

## Withdraw from Vault account

`POST /openapi/v1/mc-account/mlp/account/withdraw` — permission `TRADE`.

Withdraw from the Vault account to the fund account, then transfer into the trading account. Deducts principal first, then yield.

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| trackingId | string | Yes | Unique request tracking ID |
| symbolId | string | Yes | Symbol ID |
| amount | number | Yes | Withdrawal amount |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| data | string | `data` is a scalar of this type |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/mlp/account/withdraw" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```
