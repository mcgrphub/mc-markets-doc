## MLP账户提现

`POST /openapi/v1/mc-account/mlp/account/withdraw` — permission `TRADE`.

MLP账户提现，从MLP账户转出到资金账户，然后划入交易账户。优先扣减本金，再扣减收益

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| trackingId | string | Yes | 用于跟踪请求的唯一标识 |
| symbolId | string | Yes | 关联ac_symbol表 |
| amount | number | Yes | 提现金额 |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/mlp/account/withdraw" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```

## 查询MLP参数

`GET /openapi/v1/mc-account/mlp/config` — permission `READ`.

查询当前请求上下文业务线下的 MLP 收益率等参数，逻辑与后台 MlpConfigController#getConfig 一致，响应为 OpenAPI 专用模型。

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| id | integer | 配置ID |
| businessLineId | integer | 业务线ID |
| annualYieldRate | number | 调整收益率（年化），单位：百分比 |
| lockPeriodDays | integer | 锁仓期，单位：天 |
| singleUserDepositLimit | number | 单用户存款上限，单位：USDC |
| tvlBaseAmount | number | TVL金额基准值，单位：USDC |
| integralTierConfig | string | 积分档位配置，格式：积分最小值:存款限额比例,...，例如：50:25,300:50,1500:75,10000:100 |
| createBy | string | 创建人 |
| createTime | string | 创建时间 |
| updateBy | string | 更新人 |
| updateTime | string | 更新时间 |
| createTimeTimestamp | integer | 创建时间毫秒时间戳 |
| updateTimeTimestamp | integer | 更新时间毫秒时间戳 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/mlp/config" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```
