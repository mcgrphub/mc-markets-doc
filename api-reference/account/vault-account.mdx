| Method | Path | Permission | Description |

## Withdraw from Vault account

`POST /openapi/v1/mc-account/mlp/account/withdraw` — permission `TRADE`.

Withdraw from the Vault account to the fund account, then transfer into the trading account. Deducts principal first, then yield.

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| trackingId | string | Yes | Unique request tracking ID |
| symbolId | string | Yes | Symbol ID |
| amount | number | Yes | Withdrawal amount |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/mlp/account/withdraw" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```

## Query Vault account parameters

`GET /openapi/v1/mc-account/mlp/config` — permission `READ`.

Query Vault account yield rate and related parameters for the current business line.

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| id | integer | Config ID |
| businessLineId | integer | Business line ID |
| annualYieldRate | number | Adjusted annual yield rate (percent) |
| lockPeriodDays | integer | Lock period in days |
| singleUserDepositLimit | number | Per-user deposit limit (USDC) |
| tvlBaseAmount | number | TVL base amount (USDC) |
| integralTierConfig | string | Points tier config, format `minPoints:depositLimitPercent,...`, for example `50:25,300:50,1500:75,10000:100` |
| createBy | string | Created by |
| createTime | string | Create time |
| updateBy | string | Updated by |
| updateTime | string | Update time |
| createTimeTimestamp | integer | Create time (epoch ms) |
| updateTimeTimestamp | integer | Update time (epoch ms) |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/mlp/config" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```
