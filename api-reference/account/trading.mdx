Paths in this group use the `/openapi/v1/mc-account/` prefix.

## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| POST | /openapi/v1/mc-account/transaction/transaction/record/transactionId/&#123;id&#125; | READ | 根据transactionId查询交易记录详情 |
| POST | /openapi/v1/mc-account/transaction/transaction/record/page | READ | 分页查询交易记录 |
| POST | /openapi/v1/mc-account/transaction/exchange/v2 | TRADE | 兑换v2, 支持询价 |
| POST | /openapi/v1/mc-account/transaction/account/record/page | READ | 分页查询资金流水记录 |

<Note>
  The interactive Playground is disabled. Sign requests with AK/SK as described in [Authentication](/authentication). See [Base URL](/api-reference/base-url) for the gateway host.
</Note>

## 兑换v2, 支持询价

`POST /openapi/v1/mc-account/transaction/exchange/v2` — permission `TRADE`.

需要先调用[触发2FA验证]接口,触发2FA验证

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| trackingId | string | Yes | 用于跟踪请求的唯一标识 |
| symbol | string | Yes | 如 BTC/USDT |
| exchangeSymbol | string | Yes | 如 BTC/USDT |
| amount | number | Yes | 兑换金额/兑换币种的数量 |
| verify2FaTokens | array | No | 通过2FA验证接口获取的临时验证令牌列表，支持多种类型的2FA验证 |
| quoteSessionId | string | Yes | asw1w123a |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/transaction/exchange/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```
