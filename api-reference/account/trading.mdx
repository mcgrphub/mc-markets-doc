Paths in this group use the `/openapi/v1/mc-account/` prefix.

## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| POST | /openapi/v1/mc-account/transaction/transaction/record/transactionId/&#123;id&#125; | READ | Get transaction record details by transactionId |
| POST | /openapi/v1/mc-account/transaction/transaction/record/page | READ | Paginate transaction records |
| POST | /openapi/v1/mc-account/transaction/exchange/v2 | TRADE | Flash conversion (quote-supported) |
| POST | /openapi/v1/mc-account/transaction/account/record/page | READ | Paginate fund account ledger records |

<Note>
  The interactive Playground is disabled. Sign requests with AK/SK as described in [Authentication](/authentication). See [Base URL](/api-reference/base-url) for the gateway host.
</Note>

## Paginate transaction records

`POST /openapi/v1/mc-account/transaction/transaction/record/page` — permission `READ`.

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No | Page size |
| current | integer | No | Current page number, starting from 1 |
| orderBy | string | No | Multi-field sort; comma-separated fields |
| orderDirection | string | No | ASC ascending / DESC descending; comma-separated for multiple fields |
| trackingId | string | Yes | Unique request tracking ID |
| symbolId | integer | No | Filter by currency ID |
| exchangeSymbolId | integer | No | Filter by destination currency ID |
| transactionType | string | No | Transaction type enum |
| transactionTypes | array | No | Applies only when transactionType==null |
| status | string | No | Filter by status: SUCCESS / FAIL / PENDING |
| statuses | array | No | Applies only when status==null |
| createTimeStart | string | No | yyyy-MM-dd HH:mm:ss.SSS |
| createTimeStartTimestamp | integer | No | Unix timestamp (ms) |
| createTimeEnd | string | No | yyyy-MM-dd HH:mm:ss.SSS |
| createTimeEndTimestamp | integer | No | Unix timestamp (ms) |
| transactionMethod | string | No | Transaction method enum |

| total | integer | Total number of records |
| size | integer | Page size |

| hasPrevious | boolean | Whether a previous page exists |
| hasNext | boolean | Whether a next page exists |

## Flash conversion (quote-supported)

`POST /openapi/v1/mc-account/transaction/exchange/v2` — permission `TRADE`.

Complete a flash conversion. First call [Query flash conversion exchange rate](/api-reference/account/exchange-rate#query-flash-conversion-exchange-rate) to get a valid `quoteSessionId`.

### Request body (JSON)

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| trackingId | string | Yes | Unique request tracking ID |
| symbol | string | Yes | e.g. BTC/USDT |
| exchangeSymbol | string | Yes | e.g. BTC/USDT |
| amount | number | Yes | Exchange amount / quantity of the exchange symbol |
| quoteSessionId | string | Yes | Quote session ID from the flash conversion rate quote |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| data | string | `data` is a scalar of this type |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/transaction/exchange/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```
