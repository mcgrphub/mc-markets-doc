## Paginated funding fee history V2

`GET /openapi/v1/mc-account/funding-fee-record/page/v2` — permission `READ`.

`feeType` is fixed to INTEREST_FEE/FUNDING_FEE; amounts are aggregated by `positionId` + `dealTime`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No |  |
| current | integer | No |  |
| orderBy | string | No | Multi-field sort; comma-separated |
| orderDirection | string | No | ASC ascending / DESC descending; comma-separated for multiple fields |
| userId | string | No | User ID |
| login | string | No | MT5 login ID |
| dealTimeStart | string | No | Fee deduction start time |
| dealTimeEnd | string | No | Fee deduction end time |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| records | array |  |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/funding-fee-record/page/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1"
```
