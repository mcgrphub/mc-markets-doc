## 资金费历史记录分页查询接口V2

`GET /openapi/v1/mc-account/funding-fee-record/page/v2` — permission `READ`.

feeType固定为INTEREST_FEE/FUNDING_FEE，按positionId+dealTime分组汇总amount

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No |  |
| current | integer | No |  |
| orderBy | string | No | 支持多字段排序，用逗号分隔 |
| orderDirection | string | No | ASC升序/DESC降序，多字段时用逗号分隔 |
| userId | string | No | 用户ID |
| login | string | No | mt5登录账号 |
| dealTimeStart | string | No | 开始扣费时间 |
| dealTimeEnd | string | No | 结束扣费时间 |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| records | array |  |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/funding-fee-record/page/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1"
```
