## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| GET | /openapi/v1/mc-account/fund/account/spot/positions/v2 | READ | Query positions V2 |
| GET | /openapi/v1/mc-account/fund/account/spot-account/positions | READ | Query spot account positions |
| GET | /openapi/v1/mc-account/fund/account/por | READ | Query Proof of Reserves (PoR) |

<Note>
  The interactive Playground is disabled. Sign requests with AK/SK as described in [Authentication](/authentication). See [Base URL](/api-reference/base-url) for the gateway host.
</Note>

## Query positions V2

`GET /openapi/v1/mc-account/fund/account/spot/positions/v2` — permission `READ`.

Query positions.

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| uid | string | User ID |
| symbolInfo | object | Position information, key is the trading symbol, value is position details |

`symbolInfo` fields:

| Field | Type | Description |
| --- | --- | --- |
| <key> | object | Position information, key is the trading symbol, value is position details |

`symbolInfo.<key>` map value fields:

| Field | Type | Description |
| --- | --- | --- |
| contractSize | number | Contract Size |
| freeMargin | number | Available |
| precision | number | Precision |
| priceCurrent | number | Current Price |
| symbol | string | Trading Symbol |
| usdValue | number | USD Value |
| volumeExt | number | Extended Position Volume |
| volume | number | Actual Position Quantity |
| openPrice | number | Open Price |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/fund/account/spot/positions/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```

## Query spot account positions

`GET /openapi/v1/mc-account/fund/account/spot-account/positions` — permission `READ`.

Query spot account positions.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| spotId | string | Yes |  |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| spotId | string | Spot Account ID |
| symbolInfo | object | Position information, key is the trading symbol, value is position details |

`symbolInfo` fields:

| Field | Type | Description |
| --- | --- | --- |
| <key> | object | Position information, key is the trading symbol, value is position details |

`symbolInfo.<key>` map value fields:

| Field | Type | Description |
| --- | --- | --- |
| contractSize | number | Contract Size |
| freeMargin | number | Available |
| precision | number | Precision |
| priceCurrent | number | Current Price |
| symbol | string | Trading Symbol |
| usdValue | number | USD Value |
| rateProfit | number | P/L Exchange Rate |
| volume | number | Actual Position Quantity |
| volumeExt | number | Actual Position Quantity |
| openPrice | number | Current Price |
| freeAmount | number | Available (excluding portion locked by pending orders). This field is non-zero only for USDC |
| volumeDisplay | string | Total balance display value |
| volumeScale | integer | Total balance display precision |
| volumeExtDec | number |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/fund/account/spot-account/positions" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "spotId=VALUE"
```

## Query Proof of Reserves (PoR)

`GET /openapi/v1/mc-account/fund/account/por` — permission `READ`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| batchId | integer | No |  |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| treeBatchId | integer | Merkle Tree Build Batch |
| userId | string | User ID |
| userHash | string | PoRv2 User Hash (user_hash) |
| assetInfo | string | Asset Information |
| leafHash | string | Merkle leaf hash value |
| pathLayers | string | Recursive layer path, each layer has a group + its position within it |
| platformAssetProof | array | Platform Proof of Assets |

`platformAssetProof[]` items:

| Field | Type | Description |
| --- | --- | --- |
| chain | string | Chain Name |
| address | string | Wallet Address |
| assetInfo | string | Asset Information |
| signature | string | Signature |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/fund/account/por" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```
