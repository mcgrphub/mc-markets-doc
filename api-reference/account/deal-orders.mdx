Paths in this group use the `/openapi/v1/mc-account/` prefix.

## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| GET | /openapi/v1/mc-account/deal-order | READ | Paginate deal orders |
| GET | /openapi/v1/mc-account/deal-order/&#123;dealId&#125; | READ | Get deal order details |

<Note>
  The interactive Playground is disabled. Sign requests with AK/SK as described in [Authentication](/authentication). See [Base URL](/api-reference/base-url) for the gateway host.
</Note>

## Paginate deal orders

`GET /openapi/v1/mc-account/deal-order` — permission `READ`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No | Page size |
| current | integer | No | Current page number, starting from 1 |
| orderBy | string | No | Multi-field sort; comma-separated fields |
| orderDirection | string | No | ASC ascending / DESC descending; comma-separated for multiple fields |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| records | array | Records on the current page |
| total | integer | Total number of records |
| size | integer | Page size |
| current | integer | Current page number |
| pages | integer | Total number of pages |
| hasPrevious | boolean | Whether a previous page exists |
| hasNext | boolean | Whether a next page exists |

`records[]` items:

| Field | Type | Description |
| --- | --- | --- |
| id | integer | Transaction Record ID |
| uid | string | Associated Account Table |
| transactionId | string | Transaction ID |
| symbolId | integer | Currency ID |
| acSymbol | string | Currency Name / From |
| network | string | Network Protocol |
| exchangeSymbolId | integer | Target Currency ID |
| exchangeSymbol | string | Target Currency Name / To |
| transactionType | string | Trade Type Enum |
| status | string | Trade Status |
| amount | number | Transaction Amount |
| amountStr | string | Transaction Amount String |
| amountDisplay | string | Transaction amount display value |
| amountScale | integer | Transaction amount display precision |
| exchangeAmount | number | Target Amount |
| exchangeAmountDisplay | string | Target amount display value |
| fee | number | Trading Fee |
| feeDisplay | string | Trading fee display value |
| createTime | string | Trade Start Time |
| finishTime | string | Trade Completion Time |
| failureReason | string | Failure Reason |
| errorCode | string | Failure Error Code |
| exchangeRate | number | Exchange Rate |
| convertExchangeRate | number | Exchange rate (to/from) |
| rateDisplay | string | Exchange rate display value |
| txHash | string | Blockchain Transaction Hash |
| feeSymbol | string | Fee Currency |
| sourceAccountType | string | Source Account Type |
| targetAccountType | string | Target Account Type |

| targetAccountId | string | Target Account ID |

| targetWalletAddress | string | Target Wallet Address |
| extParams | string | Extended Parameters |
| sourceAccountBalance | number | Source Account Balance After Transfer |
| targetAccountBalance | number | Target Account Balance After Transfer |
| fundFlowType | string | Fund Flow Type |
| transactionMethod | string | Trading Method Enum |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/deal-order" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1"
```
