The MC OpenAPI is served exclusively over HTTPS through the **MC API Gateway**. Every request must target the base URL below, followed by the API path prefix (`/openapi/v1`) and the specific endpoint — for example, `https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/usd`.

## Base URL

| | Base URL |
| --- | --- |
| Production | `https://gateway.mcconnects.com` |

<Note>
  Authentication uses **AK/SK** credentials and **HMAC-SHA256** signing. Keys are issued from the MC platform.
</Note>

## Path prefix

All resource paths are versioned under `/openapi/v1/…`. Combine the gateway host with the endpoint's path:

```text
{gateway}/openapi/v1/{resource}/…
```

For example:

- `https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/usd`
- `https://gateway.mcconnects.com/openapi/v1/mc-account/finance/account/overview`

## Examples

<CodeGroup>

```bash cURL
curl -X POST https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/usd \
  -H "X-Access-Key: $MC_ACCESS_KEY" \
  -H "X-Signature: $MC_SIGNATURE" \
  -H "X-Timestamp: $MC_TIMESTAMP" \
  -H "Content-Type: application/json"
```

```python Python
import os

MC_GATEWAY = os.environ.get(
    "MC_GATEWAY",
    "https://gateway.mcconnects.com",
)

url = f"{MC_GATEWAY}/openapi/v1/mc-account/exchange-rate/query/usd"
```

```javascript Node.js
const MC_GATEWAY =
  process.env.MC_GATEWAY ?? "https://gateway.mcconnects.com";

const url = `${MC_GATEWAY}/openapi/v1/mc-account/exchange-rate/query/usd`;
```

```go Go
package main

import (
	"fmt"
	"os"
)

func main() {
	gateway := os.Getenv("MC_GATEWAY")
	if gateway == "" {
		gateway = "https://gateway.mcconnects.com"
	}
	url := fmt.Sprintf("%s/openapi/v1/mc-account/exchange-rate/query/usd", gateway)
	fmt.Println(url)
}
```

</CodeGroup>

<Warning>
  Always use HTTPS. Plain HTTP requests are rejected. If your client follows redirects, POST bodies may not be re-sent, causing signature verification to fail on the retried request.
</Warning>

<Tip>
  Store the gateway host as an environment variable (for example, `MC_GATEWAY`) in your application config for a single place to manage the host.
</Tip>
