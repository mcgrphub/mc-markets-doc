**Market data** endpoints return symbol rankings and candlestick data. All operations in this group require `READ`. Paths use the `/openapi/v1/mc-aggregator/` prefix.

## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| GET | /openapi/v1/mc-aggregator/market-data/symbol-ranking | READ | Paginated symbol ranking by category and sort fields. |
| GET | /openapi/v1/mc-aggregator/market-data/kline | READ | Candlestick data by symbol, period, and time range. |

<Note>
  The interactive Playground is disabled. Sign requests with AK/SK as described in [Authentication](/authentication). See [Base URL](/api-reference/base-url) for the gateway host.
</Note>

## Symbol ranking

`GET /openapi/v1/mc-aggregator/market-data/symbol-ranking` — permission `READ`.

Paginated symbol ranking by category and sort dimensions.

### Query parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| category | string | Yes | Tab category: CONTRACT / SPOT / CRYPTO / TRADFI |
| subGroup | string | No | TradFi secondary tab: ALL / FOREX / METALS / INDICES / ENERGY / STOCKS |
| hotOnly | boolean | No | Return only hot symbols (isHot=1 and status=1) |
| sortBy | string | No | Sort field: symbol / bid / change24hPct |
| sortOrder | string | No | Sort direction: asc / desc |
| page | integer | No | Page number, starting at 1 |
| pageSize | integer | No | Page size |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| total | integer | Total symbols in the current category |
| page | integer | Current page |
| pageSize | integer | Page size |
| snapshotVersion | integer | Ranking snapshot version |
| items | array | Ranking rows for the current page |

`items[]` fields:

| Field | Type | Description |
| --- | --- | --- |
| rank | integer | Global rank |
| symbol | string | Symbol code |
| bid | string | Bid price |
| change24hAbs | string | 24h absolute change |
| change24hPct | string | 24h change as a decimal (for example -0.0146 is -1.46%) |
| isHot | boolean | Whether the symbol is hot |
| pricingDegree | integer | Price decimal places |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-aggregator/market-data/symbol-ranking" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  --data-urlencode "category=SPOT" \
  --data-urlencode "sortBy=change24hPct" \
  --data-urlencode "sortOrder=desc" \
  --data-urlencode "page=1" \
  --data-urlencode "pageSize=20"
```

## Candlestick (K-line) data

`GET /openapi/v1/mc-aggregator/market-data/kline` — permission `READ`.

Query candlestick data by symbol, interval, and time range.

### Query parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| symbol | string | Yes | Symbol code (for example EURUSD) |
| period | integer | Yes | Interval in minutes: 1 / 5 / 15 / 60 / 240 / 1440; week = 1440*7; month = 1440*30 |
| endTime | integer | Yes | End time as a Unix timestamp in seconds |
| startTime | integer | No | Start time as a Unix timestamp in seconds |
| limit | integer | No | Max rows per response; maximum 500 |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| pricingDegree | integer | Price precision used when converting values to floats |
| klineDataList | array | Compact candles; each item is open, close, high, low, volume, then Unix seconds |
| klineDisplayList | array | Display fields aligned with klineDataList |

`klineDisplayList[]` common fields: `openStr` / `openDisplay` / `closeStr` / `closeDisplay` / `highStr` / `highDisplay` / `lowStr` / `lowDisplay` / `volumeStr` / `volumeDisplay` with matching `*Scale` values, plus `timestamp` (seconds).

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-aggregator/market-data/kline" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  --data-urlencode "symbol=EURUSD" \
  --data-urlencode "period=60" \
  --data-urlencode "startTime=1755792000" \
  --data-urlencode "endTime=1755878400" \
  --data-urlencode "limit=500"
```
