| Method | Path | Permission | Description |
| --- | --- | --- | --- |

## Fund movement list

`GET /openapi/v1/mc-trade/account/positions` — permission `READ`.

Query the trading account fund movement list (collateral increase/decrease history) for the current user. UID is taken from the `X-UID` request header.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No |  |
| current | integer | No |  |
| orderBy | string | No |  |
| sortOrder | string | No |  |
| login | string | Yes | MT5 contract account ID / spot account ID |
| from | integer | Yes |  |
| to | integer | Yes |  |
| symbol | string | No |  |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| records | array |  |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/account/positions" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1" \
  --data-urlencode "login=VALUE" \
  --data-urlencode "from=1755792000"
```

## Query current positions V2

`GET /openapi/v1/mc-trade/account/positions/current/v2` — permission `READ`.

Query all open positions for the current user (UID from the `X-UID` header). MT5 often stores floats as integers to avoid precision issues: fields without an Ext suffix use 4 decimal places (divide by 10000); fields with an Ext suffix use 8 decimal places (divide by 100000000). For example, `mt5_positions` stores volume as `Volume` and `VolumeExt`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| login | string | Yes | MT5 contract account ID / spot account ID |
| symbol | string | No | Trading symbol |
| positionType | string | No | Omit or pass ALL for all, COLLATERAL for collateral only, SYMBOL for trading positions only |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/account/positions/current/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "login=VALUE"
```

## PnL statistics for a time range

`GET /openapi/v1/mc-trade/account/pnl` — permission `READ`.

Query PnL for a time range (for example, the past week). UID is taken from the `X-UID` header. If `from` and `to` are omitted, defaults to the last seven days.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| from | integer | No |  |
| to | integer | No |  |
| login | string | Yes |  |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| uid | integer | Account login ID |
| totalProfit | number | Total profit in range |
| totalLoss | number | Total loss in range (negative value) |
| netPnl | number | Net PnL in range |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/account/pnl" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "login=VALUE"
```

## Query total long/short contract volume by trading symbol

`GET /openapi/v1/mc-trade/account/contract-volume-by-symbol` — permission `Public`.

`symbol` maps to `mt5_positions.Symbol`; only symbols whose Path starts with CFD are counted; volume = VolumeExt/1e8.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| symbol | string | Yes |  |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| symbol | string |  |
| longVolume | number |  |
| shortVolume | number |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/account/contract-volume-by-symbol" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "symbol=BTCUSDT"
```

## Query total long/short contract notional by trading symbol

`GET /openapi/v1/mc-trade/account/contract-notional-by-symbol` — permission `Public`.

`symbol` maps to `mt5_positions.Symbol`; only symbols whose Path starts with CFD are counted; notional = VolumeExt/1e8 * PriceCurrent * RateProfit * ContractSize (not differentiated by Symbol).

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| symbol | string |  |
| longNotional | number |  |
| shortNotional | number |  |

curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/account/contract-notional-by-symbol" \
