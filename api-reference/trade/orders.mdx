## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| GET | /openapi/v1/mc-trade/trade/orders/pending/v2 | READ | Query pending (unfilled) orders V2 |

## Query pending (unfilled) orders V2

`GET /openapi/v1/mc-trade/trade/orders/pending/v2` — permission `READ`.

Query pending (unfilled) orders for the current user. UID is taken from the `X-UID` request header.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| symbol | string | No | Trading symbol, optional |
| login | string | Yes | Trading account login |

### Response `data` fields

`data` is an array of objects. Element fields follow.

`data[]` items:

| Field | Type | Description |
| --- | --- | --- |
| order | integer | Order number, primary key |
| login | integer | Login account, MT5 account login number |
| dealer | integer | Dealer ID, the MT5 dealer code |
| symbol | string | Trading symbol, e.g., EURUSD, GBPUSD, etc. |
| digits | integer | Price precision, the number of decimal places for the trading symbol |
| digitsCurrency | integer | Currency precision, the number of currency decimal places for the trading symbol |
| contractSize | number | Contract size, the standard size per lot |
| state | integer | Order Status |
| reason | integer | Reason, the reason code for the order change |
| timeSetup | integer | Order time, the timestamp when the order was created, in milliseconds, 13 digits |
| timeExpiration | integer | Expiry time, the expiration timestamp of the order, in milliseconds, 13 digits |
| timeDone | integer | Completion time, the timestamp when the order was completed, in milliseconds, 13 digits |
| timeSetupMsc | integer | Order time (ms), the timestamp when the order was created (milliseconds), 13 digits |
| timeDoneMsc | integer | Completion time (ms), the timestamp when the order was completed (milliseconds), 13 digits |
| modifyFlags | integer | Modification flags, the flag bits for order modification |
| type | integer | Order type, e.g., buy, sell, etc. |
| typeFill | integer | Deal type, the fill type of the order |
| typeTime | integer | Time validity type, the validity type of the order |
| priceOrder | number | Order price, the execution price of the order |
| priceOrderDisplay | string | Order price display value |
| priceOrderScale | integer | Order price display decimal places |
| priceTrigger | number | Trigger price, the trigger price of the order |
| priceTriggerDisplay | string | Trigger price display value |
| priceTriggerScale | integer | Trigger price display decimal places |
| priceCurrent | number | Current price, the current market price of the order |
| priceCurrentDisplay | string | Current price display value |
| priceCurrentScale | integer | Current price display decimal places |
| priceSlDisplay | string | Stop loss price display value |
| priceSlScale | integer | Stop loss price display decimal places |
| priceTpDisplay | string | Take profit price display value |
| priceTpScale | integer | Take profit price display decimal places |
| volumeInitialExt | number | Initial extended volume, the initial extended volume of the order (for large orders) |
| volumeCurrentExt | number | Current extended volume, the current extended volume of the order |
| quantity | number | User quantity, equals current remaining volumeCurrentExt divided by 10^8 |
| rateMargin | number | Margin exchange rate, the exchange rate used when calculating margin |
| activationMode | integer | Activation mode, the activation mode of the pending order |
| activationTime | integer | Activation time, the activation timestamp of the pending order |
| activationPrice | number | Activation price, the activation price of the pending order |
| activationFlags | integer | Activation flag, the flag bit for pending order activation |
| volumeInitial | number | Initial volume, the initial trading volume of the order |
| volumeInitialDisplay | string | Initial volume display value |
| volumeInitialScale | integer | Initial volume display decimal places |
| volumeCurrent | number | Current volume, the current trading volume of the order |
| volumeCurrentDisplay | string | Current volume display value |
| volumeCurrentScale | integer | Current volume display decimal places |
| externalID | string | External ID, the ID associated with the external system |
| priceSL | number | Stop loss price, the set stop loss level |
| priceTP | number | Take profit price, the set take profit level |
| expertID | integer | Expert Advisor ID. If the order was created by an EA (Expert Advisor), records the EA ID |
| positionID | integer | Position ID, the associated position ID |
| positionByID | integer | Opposite position ID, the associated opposite position ID |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/orders/pending/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "login=VALUE"
```

## Query historical orders V3

`GET /openapi/v1/mc-trade/trade/orders/history/v3` — permission `READ`.

Query historical orders by UID and account data type (SPOT/CONTRACT/SPOT_CONTRACT), including spot and contract data.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No | Page size |
| current | integer | No | Current page number |
| orderBy | string | No | Sort fields, comma-separated |
| sortOrder | string | No | Sort direction: ASC / DESC |
| type | string | Yes | Account data type: SPOT — spot, CONTRACT — contract, SPOT_CONTRACT — spot and contract combined |
| from | integer | Yes | Start timestamp (Unix timestamp, seconds) |
| to | integer | Yes | End timestamp (Unix timestamp, seconds) |
| symbol | string | No | Trading symbol name |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| records | array | Records on the current page |
| total | integer | Total number of records |
| size | integer | Page size |
| current | integer | Current page number |
| pages | integer | Total number of pages |
| hasPrevious | boolean | Whether a previous page exists |
| hasNext | boolean | Whether a next page exists |
