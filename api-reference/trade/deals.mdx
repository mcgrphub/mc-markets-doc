## Query historical deals V3

`GET /openapi/v1/mc-trade/trade/deals/history/v3` — permission `READ`.

Query historical deals by UID and account data type (SPOT/CONTRACT/SPOT_CONTRACT).

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No |  |
| current | integer | No |  |
| orderBy | string | No |  |
| sortOrder | string | No |  |
| type | string | Yes | Account data type: SPOT — spot, CONTRACT — contract, SPOT_CONTRACT — spot and contract combined |
| from | integer | Yes | Start timestamp (Unix timestamp, seconds) |
| to | integer | Yes | End timestamp (Unix timestamp, seconds) |
| symbol | string | No | Trading symbol name |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| records | array |  |
| records[].deal | integer | Deal number, primary key |
| records[].login | integer | Login account, MT5 account login number |
| records[].dealer | integer | Dealer ID, the MT5 dealer code |
| records[].order | integer | Order number, associated order number |
| records[].action | integer | 0-Buy, 1-Sell |
| records[].entry | integer | 0-Open, 1-Close |
| records[].reason | integer | Reason, the reason code for the deal change |
| records[].digits | integer | Price precision, the number of decimal places for the trading symbol |
| records[].digitsCurrency | integer | Currency precision, the number of currency decimal places for the trading symbol |
| records[].contractSize | number | Contract size, the standard size per lot |
| records[].time | integer | Deal time, the timestamp when the deal occurred, in milliseconds, 13 digits |
| records[].timeMsc | integer | Deal time (ms), the timestamp when the deal occurred (milliseconds), 13 digits |
| records[].symbol | string | Trading symbol, e.g., EURUSD, GBPUSD, etc. |
| records[].price | number | Deal price, the execution price of the deal |
| records[].volumeExt | integer | Extended volume, the extended volume of the deal (for large orders). Divide by 100000000 to convert to a floating point number |
| records[].quantity | number | User quantity, volumeExt divided by 10^8 |
| records[].profit | number | Profit/Loss, the profit/loss amount generated by the deal |
| records[].storage | number | Storage fee, the storage fee generated by the position |
| records[].commission | number | Commission, the commission fee generated by the deal |
| records[].commissionDisplay | string | Commission display value, rounded using CEILING with sign, fixed to 6 decimal places |
| records[].commissionScale | integer | Commission display decimal places |
| records[].profitDisplay | string | Profit/Loss display value, fixed to 8 decimal places using HALF_UP rounding |
| records[].profitScale | integer | Profit/Loss display decimal places |
| records[].fee | number | Commission, the commission fee generated by the deal |
| records[].rateProfit | number | Profit/Loss exchange rate, the exchange rate used when calculating P/L |
| records[].rateMargin | number | Margin exchange rate, the exchange rate used when calculating margin |
| records[].profitRaw | number | Original profit/loss, the raw profit/loss amount of the deal (before exchange rate conversion) |
| records[].pricePosition | number | Position price, the average price of the position |
| records[].volumeClosedExt | integer | Closed extended volume, the closed extended volume |
| records[].tickValue | number | Point value, the value per point |
| records[].tickSize | number | Point size, the size per point |
| records[].flags | integer | Flags, the flag bits of the deal |
| records[].value | number | Value, the value of the deal |
| records[].gateway | string | Gateway, the name of the dealing gateway |
| records[].priceGateway | number | Gateway price, the dealing price at the gateway |
| records[].modifyFlags | integer | Modification flags, the flag bits for deal modification |
| records[].marketBid | number | Market bid price, the current market bid price |
| records[].marketAsk | number | Market ask price, the current market ask price |
| records[].marketLast | number | Market last price, the last market dealing price |
| records[].volume | integer | Volume, the trading volume of the deal in lots. Divide by 10000 to convert to a floating point number |
| records[].volumeClosed | integer | Closed volume, the closed trading volume |
| records[].externalID | string | External ID, the ID associated with the external system |
| records[].expertID | integer | Expert Advisor ID. If the deal was created by an EA (Expert Advisor), records the EA ID |
| records[].positionID | integer | Position ID, the associated position ID |
| records[].priceSL | number | Stop loss price, the set stop loss level |
| records[].priceTP | number | Take profit price, the set take profit level |
| records[].on_chain_status | string | On-chain status (associated with ac_deal_order) |
| records[].tx_hash | string | On-chain transaction hash (associated with ac_deal_order) |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/deals/history/v3" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1" \
  --data-urlencode "type=VALUE" \
  --data-urlencode "from=1755792000"
```

## Query recent deals by symbol

`GET /openapi/v1/mc-trade/trade/deals/history/recent` — permission `Public`.

Query recent deals by symbol; result count is controlled by `limit`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| symbol | string | Yes |  |
| limit | integer | Yes |  |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| data | array | `data` is an array of objects; element fields use `[]...` |
| [].action | integer | 0-Buy, 1-Sell |
| [].price | number | Deal Price |
| [].volumeExt | integer | Extended Volume |
| [].timeMsc | integer | Deal Time (ms) |
| [].orderId | integer | Order Number |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/deals/history/recent" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "symbol=BTCUSDT" \
  --data-urlencode "limit=1"
```
