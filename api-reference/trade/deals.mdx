## 查询历史成交列表V3

`GET /openapi/v1/mc-trade/trade/deals/history/v3` — permission `READ`.

根据 uid 和账户数据类型(SPOT/CONTRACT/SPOT_CONTRACT)查询历史成交列表

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No |  |
| current | integer | No |  |
| orderBy | string | No |  |
| sortOrder | string | No |  |
| type | string | Yes | 账户数据类型：SPOT-现货，CONTRACT-合约，SPOT_CONTRACT-现货与合约合并 |
| from | integer | Yes | 开始时间戳（Unix时间戳，秒） |
| to | integer | Yes | 结束时间戳（Unix时间戳，秒） |
| symbol | string | No | 交易品种名 |

### Response `data` fields (summary)

| Field | Type | Description |
| --- | --- | --- |
| records | array |  |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/deals/history/v3" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1" \
  --data-urlencode "type=VALUE" \
  --data-urlencode "from=1755792000"
```

## 获取品种最近成交列表

`GET /openapi/v1/mc-trade/trade/deals/history/recent` — permission `公开访问`.

根据 symbol 查询最近成交，返回条数由 limit 控制

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| symbol | string | Yes |  |
| limit | integer | Yes |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/deals/history/recent" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "symbol=BTCUSDT" \
  --data-urlencode "limit=1"
```
