The **Funding Rate (资金费率)** group queries funding-rate history for perpetual contracts and similar products. Unlike account-side [Funding Fee History](/api-reference/endpoints/funding-fee) (user position fee ledger), this group returns market funding-rate time series.

## Endpoints

| Method | Path | Permission | Description |
| --- | --- | --- | --- |
| GET | `/openapi/v1/funding-rate` | `READ` | Paginated query of funding-rate history for a symbol. |

---

## Paginated query — funding-rate history

`GET /openapi/v1/funding-rate` — permission `READ`.

### Query parameters — FundingRatePageReq

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `symbol` | string | Yes | Trading symbol |
| `isSettlementRate` | boolean | No | Settlement rate flag; default `true` |
| `current` / `size` etc. | — | No | Standard pagination fields |

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/funding-rate?symbol=BTCUSDT&isSettlementRate=true&current=1&size=20"
```

### Response

`data` is a paginated list of `FundingRateDto`.
