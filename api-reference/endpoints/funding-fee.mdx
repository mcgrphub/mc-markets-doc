The **Funding Fee History (资金费历史记录)** group contains the v2 pagination endpoint that returns funding-fee and interest-fee records aggregated by `positionId + dealTime`. All endpoints require the `READ` permission.

## Endpoint

| Method | Path | Description |
| --- | --- | --- |
| GET | `/openapi/v1/funding-fee-record/page/v2` | Paginated query. `feeType` is fixed to `INTEREST_FEE` / `FUNDING_FEE`; records are grouped by `positionId + dealTime`, and `amount` is aggregated. |

## Paginated query (v2)

`GET /openapi/v1/funding-fee-record/page/v2` — permission `READ`.

### Query params — FundingFeeRecordPageReq

The `req` query parameter is an object; when calling this endpoint from HTTP, pass its fields as flat query parameters (`?size=10&current=1&...`).

curl -s \
  -G \
  --data-urlencode "size=20" \
  --data-urlencode "current=1" \
