Endpoints tagged **Fund Account (资金账户)** expose the user's spot positions and Proof-of-Reserves snapshots. All endpoints in this group require the `READ` permission.

## Endpoints

| Method | Path | Description |
| --- | --- | --- |
| GET | `/openapi/v1/fund/account/spot-account/positions` | Get spot account positions by `spotId`. |
| GET | `/openapi/v1/fund/account/spot/positions/v2` | Get spot position information (v2). |
| GET | `/openapi/v1/fund/account/por` | Query the user's PoRv2 Merkle proof snapshot. |

---

## Query positions by spot account

`GET /openapi/v1/fund/account/spot-account/positions` — permission `READ`.

### Query parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `spotId` | string | Yes | Spot account ID |

### Response `data` — SpotAccountPositionDTO

Includes `spotId` and `symbolInfo` (position details mapped by trading symbol).

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/fund/account/spot-account/positions?spotId=SPOT_001"
```

## Proof of Reserves (PoRv2)

`GET /openapi/v1/fund/account/por` — permission `READ`.

### Response `data` — UserAssetMerkleProofSnapshotDTO

```json
{
  "code": "0",
  "message": "success",
  "success": true,
  "data": {
    "batchId": "2024010100",
    "uid": "u_1122334455",
    "merkleRoot": "0xabc...",
    "userLeafHash": "0xdef...",
    "userTotalUsd": 15234.02,
    "proofPath": [
      { "position": "LEFT", "hash": "0x111..." },
      { "position": "RIGHT", "hash": "0x222..." }
    ],
    "assets": [
      { "symbol": "USDT", "balance": 5000.0, "usdValue": 5000.0 },
      { "symbol": "BTC", "balance": 0.15, "usdValue": 10234.02 }
    ]
  }
}
```

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/fund/account/por?batchId=2024010100"
```
