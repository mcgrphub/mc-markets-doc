## Trading symbol list

`GET /openapi/v1/mc-risk/trade/symbol/list` — permission `READ`.

Query the trading symbol list for the current user.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No |  |
| current | integer | No |  |
| orderBy | string | No | Multi-field sort; comma-separated |
| orderDirection | string | No | ASC ascending / DESC descending; comma-separated for multiple fields |
| group | string | No |  |

### Response `data` fields

| Field | Type | Description |
| --- | --- | --- |
| records | array |  |
| records[].symbol | string | Product Name |
| records[].group | string | Product Group |
| records[].flags | string | Symbol tags: MORTGAGE: Collateral, TRADE: Trading Symbol, SPOT: Spot |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-risk/trade/symbol/list" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1"
```
