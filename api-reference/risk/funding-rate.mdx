## Paginated funding rate history by trading symbol

`GET /openapi/v1/mc-risk/funding-rate` — permission `READ`.

### Query / path parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| size | integer | No |  |
| current | integer | No |  |
| orderBy | string | No | Multi-field sort; comma-separated |
| orderDirection | string | No | ASC ascending / DESC descending; comma-separated for multiple fields |
| symbol | string | Yes | Trading symbol |
| isSettlementRate | boolean | No | Whether this is the settlement rate |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-risk/funding-rate" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1" \
  --data-urlencode "symbol=BTCUSDT"
```
