你可以用它查询交易与资金流水、管理理财账户的申购与赎回、查询币种与网络信息、获取汇率与闪兑报价,以及读取持仓与储备金证明数据。

所有请求都通过 MC **API 网关** 路由,并使用 **Access Key / Secret Key** 密钥对配合 **HMAC-SHA256** 完成请求签名。

## 三步开始

<Steps>
  <Step title="获取 API Key">
    登录 MC 平台,进入 **API Key 管理**,创建新的 Key 并勾选所需权限(`READ`、`TRADE`、`WITHDRAW`)。复制 Access Key 与 Secret Key —— Secret Key 仅显示一次。
  </Step>
  <Step title="对请求签名">
    构造待签名字符串（含 `X-Nonce`，必要时含 body SHA-256），用 Secret Key 计算 HMAC-SHA256（Base64），连同 Access Key、毫秒时间戳与 nonce 一起发送。详见 [认证](/zh/authentication)。
  </Step>
  <Step title="调用接口">
    先调用 `GET /openapi/v1/mc-account/exchange-rate/query/usd` 验证集成，再浏览 [完整 API 参考](/zh/api-reference/overview)。
  </Step>
</Steps>
