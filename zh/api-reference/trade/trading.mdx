## 修改持仓

`PUT /openapi/v1/mc-trade/trade/positions/modify` — 权限 `TRADE`。

### 请求体（JSON）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| login | string | 是 | MT5合约账户ID/现货账户ID |
| order | integer | 是 | 订单ID |
| symbol | string | 是 | 交易对，例如 BTCUSDT, 仅下单有效 |
| type | integer | 是 | 订单类型 "0-OP_BUY: 市价买入", "1-OP_SELL: 市价卖出", "2-OP_BUY_LIMIT: 限价买入挂单", "3-OP_SELL_LIMIT: 限价卖出挂单", "4-OP_BUY_STOP: 止损买入挂单", "5-OP_SELL_STOP: 止损卖出挂单", "6-OP_BUY_STOP_LIMIT: 止损限价买入挂单"… |
| priceSL | number | 否 | 新的止损价格 |
| priceTP | number | 否 | 新的止盈价格 |

## 订单操作：改单

`PUT /openapi/v1/mc-trade/trade/orders/modify` — 权限 `TRADE`。

### 请求体（JSON）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| login | string | 是 | MT5合约账户ID/现货账户ID |
| symbol | string | 是 | 交易对，例如 BTCUSDT, 仅下单有效 |
| price | number | 是 | 下单价格 |
| type | integer | 是 | 订单类型 "0-OP_BUY: 市价买入", "1-OP_SELL: 市价卖出", "2-OP_BUY_LIMIT: 限价买入挂单", "3-OP_SELL_LIMIT: 限价卖出挂单", "4-OP_BUY_STOP: 止损买入挂单", "5-OP_SELL_STOP: 止损卖出挂单", "6-OP_BUY_STOP_LIMIT: 止损限价买入挂单"… |
| typeTime | integer | 是 | 有效期类型，不传默认 0。取值："0-取消前有效", "1-当天有效", "2-指定时间", "3-指定某日。到期时间为指定某日的00:00或最近的交易时间" |
| expiration | integer | 是 | 过期时间戳（typeTime=2时必填） |
| priceTrigger | number | 是 | 止损限价单触发价格 |
| priceSL | number | 是 | 止损 |
| priceTP | number | 是 | 止盈, |
| order | integer | 是 | 订单ID |
| volumeExt | integer | 是 | 交易量，10e8放大后的值, 如下单1手则传入1*100000000, 需与下单时传入的volumeExt一致 |
