## 修改持仓

`PUT /openapi/v1/mc-trade/trade/positions/modify` — 权限 `TRADE`。

### 请求体（JSON）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| login | string | 是 | MT5合约账户ID/现货账户ID |
| order | integer | 是 | 订单ID |
| symbol | string | 是 | 交易对，例如 BTCUSDT, 仅下单有效 |
| type | integer | 是 | 订单类型 "0-OP_BUY: 市价买入", "1-OP_SELL: 市价卖出", "2-OP_BUY_LIMIT: 限价买入挂单", "3-OP_SELL_LIMIT: 限价卖出挂单", "4-OP_BUY_STOP: 止损买入挂单", "5-OP_SELL_STOP: 止损卖出挂单", "6-OP_BUY_STOP_LIMIT: 止损限价买入挂单"… |
| priceSL | number | 否 | 新的止损价格 |
| priceTP | number | 否 | 新的止盈价格 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| code | string | MT5返回状态码 |
| orderId | string | MT5订单ID(如果有) |
| params | object | MT5返回参数(透传给前端或者做业务逻辑处理) |

```bash
curl -sS -X PUT "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/positions/modify" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```

## 获取预估滑点

`GET /openapi/v1/mc-trade/trade/slippage/estimate` — 权限 `READ`。

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| login | string | 是 | MT5合约账户ID/现货账户ID |
| type | integer | 是 | 订单类型 "0-OP_BUY: 市价买入", "1-OP_SELL: 市价卖出", "2-OP_BUY_LIMIT: 限价买入挂单", "3-OP_SELL_LIMIT: 限价卖出挂单", "4-OP_BUY_STOP: 止损买入挂单", "5-OP_SELL_STOP: 止损卖出挂单", "6-OP_BUY_STOP_LIMIT: 止损限价买入挂单", "7-OP_SELL_STOP_LIMIT: 止损限价卖出挂单", "8-OP_CLOSE_BY: 相互平仓订单，该操作仅适用于锁仓模式" |
| volumeExt | integer | 是 | 下单数量（*100000000扩展精度）， 如用户下单量是0.01， 则传入0.01*100000000 |
| symbol | string | 是 | 交易对/品种 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| estimatedSlippage | number | 预估滑点 |

```bash

  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"

  --data-urlencode "type=1" \

  --data-urlencode "symbol=BTCUSDT"
```
