## 查询历史成交列表V3

`GET /openapi/v1/mc-trade/trade/deals/history/v3` — 权限 `READ`。

根据 uid 和账户数据类型(SPOT/CONTRACT/SPOT_CONTRACT)查询历史成交列表

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| size | integer | 否 |  |
| current | integer | 否 |  |
| orderBy | string | 否 |  |
| sortOrder | string | 否 |  |
| type | string | 是 | 账户数据类型：SPOT-现货，CONTRACT-合约，SPOT_CONTRACT-现货与合约合并 |
| from | integer | 是 | 开始时间戳（Unix时间戳，秒） |
| to | integer | 是 | 结束时间戳（Unix时间戳，秒） |
| symbol | string | 否 | 交易品种名 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| records | array | 当前页记录列表 |
| total | integer | 总记录数 |
| size | integer | 每页条数 |
| current | integer | 当前页码 |
| pages | integer | 总页数 |
| hasPrevious | boolean | 是否有上一页 |
| hasNext | boolean | 是否有下一页 |

`records[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| order | integer | 订单号，主键 |
| login | integer | 交易账户登录号 |
| dealer | integer | 交易商ID |
| symbol | string | 交易品种，如EURUSD、GBPUSD等 |
| digits | integer | 价格精度，交易品种的价格小数位数 |
| digitsCurrency | integer | 货币精度，交易品种的货币小数位数 |
| contractSize | number | 合约大小，每手合约的标准大小 |
| state | integer | 订单状态 |
| reason | integer | 原因，订单变更的原因代码 |
| timeSetup | integer | 下单时间，订单创建的时间戳，毫秒级，长度13位 |
| timeExpiration | integer | 到期时间，订单的到期时间戳，毫秒级，长度13位 |
| timeDone | integer | 完成时间，订单完成的时间戳，毫秒级，长度13位 |
| timeSetupMsc | integer | 下单时间（毫秒），订单创建的时间戳（毫秒），毫秒级，长度13位 |
| timeDoneMsc | integer | 完成时间（毫秒），订单完成的时间戳（毫秒），毫秒级，长度13位 |
| modifyFlags | integer | 修改标志，订单修改的标志位 |
| type | integer | 订单类型，如买入、卖出等 |
| typeFill | integer | 成交类型，订单的填充方式 |
| typeTime | integer | 有效时间类型，订单的有效期类型 |
| priceOrder | number | 订单价格，订单的执行价格 |
| priceOrderStr | string | 订单价格原始字符串 |
| priceOrderDisplay | string | 订单价格展示值 |
| priceOrderScale | integer | 订单价格展示小数位 |
| priceTrigger | number | 触发价格，订单的触发价格 |
| priceTriggerDisplay | string | 触发价格展示值 |
| priceTriggerScale | integer | 触发价格展示小数位 |
| priceCurrent | number | 当前价格，订单的当前市场价格 |
| priceCurrentDisplay | string | 当前价格展示值 |
| priceCurrentScale | integer | 当前价格展示小数位 |
| priceSlDisplay | string | 止损价格展示值 |
| priceSlScale | integer | 止损价格展示小数位 |
| priceTpDisplay | string | 止盈价格展示值 |
| priceTpScale | integer | 止盈价格展示小数位 |
| volumeInitialExt | number | 初始交易量（扩展精度） |
| quantity | number | 用户数量，初始扩展交易量经持仓量精度换算后的值 |
| volumeInitialDisplay | string | 初始手数展示值 |
| volumeInitialScale | integer | 初始手数展示小数位 |
| volumeCurrentExt | number | 当前交易量（扩展精度） |
| volumeCurrentDisplay | string | 当前手数展示值 |
| volumeCurrentScale | integer | 当前手数展示小数位 |
| rateMargin | number | 保证金汇率，计算保证金时使用的汇率 |
| activationMode | integer | 激活模式，挂单的激活模式（如挂单激活方式） |
| activationTime | integer | 激活时间，挂单的激活时间戳 |
| activationPrice | number | 激活价格，挂单的激活价格 |
| activationFlags | integer | 激活标志，挂单激活的标志位 |
| volumeInitial | number | 初始手数，订单的初始交易手数 |
| volumeCurrent | number | 当前手数，订单的当前交易手数 |
| externalID | string | 外部ID，外部系统关联的ID |
| priceSL | number | 止损价格，设置的止损价位 |
| priceTP | number | 止盈价格，设置的止盈价位 |
| expertID | integer | 专家顾问ID，如果订单由 EA 创建，则记录 EA 的 ID |
| positionID | integer | 持仓ID，关联的持仓ID |
| positionByID | integer | 反向持仓ID，关联的反向持仓ID |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/deals/history/v3" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1" \
  --data-urlencode "type=VALUE" \
  --data-urlencode "from=1755792000"
```

## 获取品种最近成交列表

`GET /openapi/v1/mc-trade/trade/deals/history/recent` — 权限 `公开访问`。

根据 symbol 查询最近成交，返回条数由 limit 控制

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| symbol | string | 是 |  |
| limit | integer | 是 |  |

### 响应 `data` 字段

`data` 为对象数组。元素字段见下表。

`data[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| action | integer | 0-买入, 1-卖出 |
| price | number | 成交价格 |
| volumeExt | integer | 成交量（扩展精度） |
| timeMsc | integer | 成交时间（毫秒） |
| orderId | integer | 订单号 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-trade/trade/deals/history/recent" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "symbol=BTCUSDT" \
  --data-urlencode "limit=1"
```
