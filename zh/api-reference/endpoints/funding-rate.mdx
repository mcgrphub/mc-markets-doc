标签 **Funding Rate（资金费率）** 用于查询永续合约等品种的资金费率历史。与账户侧 [资金费历史](/zh/api-reference/endpoints/funding-fee)(用户仓位费用流水)不同,本组返回市场资金费率序列。

## 接口列表

| 方法 | 路径 | 权限 | 描述 |
| --- | --- | --- | --- |
| GET | `/openapi/v1/funding-rate` | `READ` | 分页查询指定品种的资金费率历史。 |

---

## 分页查询资金费率历史

`GET /openapi/v1/funding-rate` — 权限 `READ`。

### 查询参数 — FundingRatePageReq

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `symbol` | string | 是 | 交易币种 |
| `isSettlementRate` | boolean | 否 | 是否结算价,默认 `true` |
| `current` / `size` 等 | — | 否 | 标准分页字段 |

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/funding-rate?symbol=BTCUSDT&isSettlementRate=true&current=1&size=20"
```

### 响应

`data` 为分页的 `FundingRateDto` 列表。
