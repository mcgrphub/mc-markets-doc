标签 **Finance Account（理财账户）** 下的接口用于查询理财产品与持仓,并管理活期/定期申购与赎回。

## 接口列表

| 方法 | 路径 | 权限 | 描述 |
| --- | --- | --- | --- |
| GET | `/openapi/v1/finance/products/catalog` | `READ` | 按类型查询已启用产品目录(全量,不分页)。 |
| GET | `/openapi/v1/finance/account/positions/holding` | `READ` | 分页查询当前持仓(活期同产品多笔汇总为一行)。 |
| GET | `/openapi/v1/finance/account/positions/history` | `READ` | 分页查询用户赎回明细。 |
| GET | `/openapi/v1/finance/account/overview` | `READ` | C 端理财概览:合计、产品列表、产品下用户余额。 |
| POST | `/openapi/v1/finance/account/deposit` | `TRADE` | 从现货 → 资金账户 → 理财产品的申购。 |
| POST | `/openapi/v1/finance/account/withdraw` | `TRADE` | 将**活期**理财赎回至现货。 |
| POST | `/openapi/v1/finance/account/withdraw/fixed` | `TRADE` | 赎回**定期**持仓(本金 \+ 利息)。 |

<Note>
  活期/定期赎回权限为 `TRADE`(不是 `WITHDRAW`)。
</Note>

## 理财产品目录

`GET /openapi/v1/finance/products/catalog` — 权限 `READ`。

按类型(`DEMAND` / `FIXED`)返回已启用产品全量列表,按基础年化 \+ 超额奖励年化降序。

### 查询参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `type` | string | 是 | `DEMAND`(活期) 或 `FIXED`(定期) |

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/products/catalog?type=DEMAND"
```

### 响应 `data` — FinanceProductCatalogRsp[]

主要字段:`productCode`、`productName`、`type`、`currency`、`annualRate`、`rewardAnnualRate`、`maxAnnualRate`、`periodDays`、`minDepositAmount`、`minRedeemAmount`、`digit`、`subscribedAmount`、`fundPoolAmount`、`availableSubscriptionBalance`、`subscriptionStartTime` / `subscriptionEndTime`(及毫秒时间戳)、`maxDepositAmount`、`dailyMaxRedeemAmount`。

## 当前持仓

`GET /openapi/v1/finance/account/positions/holding` — 权限 `READ`。

分页查询持有中/已到期持仓。活期同产品多笔汇总为一行(`positionId` 可能为 `null`)。列表按 `product_code`、`currency` 升序。

`positionAmount` 为本金合计(不含未结利息;利息见 `currentRewardAmount`)。`cumulativeRewardAmount` 为历史结息原币种合计,不折 USDC。

### 查询参数

标准分页字段:`size`、`current`、`orderBy`、`orderDirection` 等(见 [分页](/zh/concepts/pagination))。

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/positions/holding?current=1&size=20"
```

## 赎回明细

`GET /openapi/v1/finance/account/positions/history` — 权限 `READ`。

按赎回动账主单分页(`EARN_WITHDRAWAL`),含产品代码、类型、持仓关联与赎回金额/时间。

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/positions/history?current=1&size=20"
```

## 申购理财产品

`POST /openapi/v1/finance/account/deposit` — 权限 `TRADE`。

将用户现货资产划入资金账户,再申购指定理财产品。

### 请求体 — FinanceDepositReq

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-dep-0001", "productCode": "USDT_CURRENT", "amount": 100 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/deposit"
```

成功时 `data` 为交易 ID 字符串。

## 赎回 — 活期产品

`POST /openapi/v1/finance/account/withdraw` — 权限 `TRADE`。

仅支持活期产品。定期持仓请使用下方定期赎回接口。

### 请求体 — FinanceWithdrawReq

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-wd-0001", "productCode": "USDT_CURRENT", "amount": 50 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/withdraw"
```

## 赎回 — 定期持仓

`POST /openapi/v1/finance/account/withdraw/fixed` — 权限 `TRADE`。

按 `positionId` 赎回定期持仓(持有中或已到期)。后端按本金 \+ 利息全额赎回。

### 请求体 — FinanceFixedWithdrawReq

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-wdf-0001", "positionId": 1234567 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/withdraw/fixed"
```

## 理财概览(C 端)

`GET /openapi/v1/finance/account/overview` — 权限 `READ`。

产品按 `product_code` 升序、`currency` 升序返回。用户侧金额来自每个产品下的持仓汇总。`totalDepositValue` 为持仓本金 × 预言机价格(USDC 计价)之和。`totalInterestValue` 与 `yesterdayInterestValue` 来自利息结算记录,换算为 USDC。`userSpotAvailableBalance` 按现货币种展示,同币种的产品共享。

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/overview"
```
