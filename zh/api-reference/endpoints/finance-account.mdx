标签 **Finance Account（理财账户）** 下的接口用于管理活期与定期理财产品余额,并提供 C 端理财概览。

## 接口列表

| 方法 | 路径 | 权限 | 描述 |
| --- | --- | --- | --- |
| POST | `/openapi/v1/finance/account/deposit` | `TRADE` | 从现货 → 资金账户 → 理财产品的申购。 |
| POST | `/openapi/v1/finance/account/withdraw` | `WITHDRAW` | 将**活期**理财赎回至现货。 |
| POST | `/openapi/v1/finance/account/withdraw/fixed` | `WITHDRAW` | 赎回**定期**持仓(本金 \+ 利息)。 |
| GET | `/openapi/v1/finance/account/overview` | `READ` | C 端理财概览:合计、产品列表、产品下用户余额。 |

---

## 申购理财产品

`POST /openapi/v1/finance/account/deposit` — 权限 `TRADE`。

将用户现货资产划入资金账户,再申购指定理财产品。

### 请求体 — FinanceDepositReq

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-dep-0001", "productCode": "USDT_CURRENT", "amount": 100 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/deposit"
```

### 响应

```json
{
  "code": "0",
  "message": "success",
  "success": true,
  "data": {
    "transactionId": "TX-DEP-0001",
    "status": "SUCCESS",
    "productCode": "USDT_CURRENT",
    "amount": 100.0
  }
}
```

## 赎回 — 活期产品

`POST /openapi/v1/finance/account/withdraw` — 权限 `WITHDRAW`。

仅支持活期产品。定期持仓请使用下方定期赎回接口。

### 请求体 — FinanceWithdrawReq

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-wd-0001", "productCode": "USDT_CURRENT", "amount": 50 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/withdraw"
```

---

## 赎回 — 定期持仓

`POST /openapi/v1/finance/account/withdraw/fixed` — 权限 `WITHDRAW`。

按 `positionId` 赎回定期持仓(持有中或已到期)。后端按本金 \+ 利息全额赎回,流程与活期赎回相一致。

### 请求体 — FinanceFixedWithdrawReq

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-wdf-0001", "positionId": 1234567 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/withdraw/fixed"
```

### 响应

```json
{ "code": "0", "message": "success", "success": true, "data": { "transactionId": "TX-WDFIX-0001", "positionId": 1234567, "status": "SUCCESS" } }
```

## 理财概览(C 端)

`GET /openapi/v1/finance/account/overview` — 权限 `READ`。

产品按 `product_code` 升序、`currency` 升序返回。用户侧金额来自每个产品下的持仓汇总。`totalDepositValue` 为持仓本金 × 预言机价格(USDC 计价)之和。`totalInterestValue` 与 `yesterdayInterestValue` 来自利息结算记录,换算为 USDC。`userSpotAvailableBalance` 按现货币种展示,同币种的产品共享。未登录时所有用户侧字段为 0。

### 响应 `data` — FinanceOverviewRsp

```json
{
  "code": "0",
  "message": "success",
  "success": true,
  "data": {
    "totalDepositValue": 12500.55,
    "totalInterestValue": 88.42,
    "yesterdayInterestValue": 1.20,
    "products": [
      {
        "productCode": "USDT_CURRENT",
        "currency": "USDT",
        "productType": "DEMAND",
        "apy": 0.045,
        "userDepositAmount": 5000.0,
        "userInterestAmount": 12.34,
        "userSpotAvailableBalance": 200.0,
        "minDepositAmount": 1.0,
        "minRedeemAmount": 1.0
      }
    ]
  }
}
```

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/finance/account/overview"
```
