标签 **MLP** 下的接口用于 MLP 账户存款/提现,以及资产、金库与收益率相关查询。

## 接口列表

| 方法 | 路径 | 权限 | 描述 |
| --- | --- | --- | --- |
| POST | `/openapi/v1/mlp/account/deposit` | `TRADE` | MLP 存款(交易账户 → 资金账户 → MLP)。 |
| POST | `/openapi/v1/mlp/account/withdraw` | `TRADE` | MLP 提现(优先扣本金,再扣收益)。 |
| GET | `/openapi/v1/mlp/account/assets` | `READ` | 查询 MLP 账户资产详情。 |
| GET | `/openapi/v1/mlp/core/stats` | `READ` | 获取 MLP 统计信息。 |
| GET | `/openapi/v1/mlp/yesterday-yield-rate-data` | `READ` | MLP 昨日回报率图表数据。 |
| GET | `/openapi/v1/mlp/tvl-data` | `READ` | MLP 金库 TVL 图表数据。 |
| GET | `/openapi/v1/mlp/apr-data` | `READ` | MLP APR 图表数据。 |
| GET | `/openapi/v1/mlp/treasury-info` | `READ` | MLP 金库相关信息。 |
| GET | `/openapi/v1/mlp/deposit-info` | `READ` | MLP 存款信息。 |
| GET | `/openapi/v1/mlp/withdraw-info` | `READ` | MLP 提款信息。 |
| GET | `/openapi/v1/mlp/config` | `READ` | 查询当前业务线 MLP 参数(年化、锁仓期、限额等)。 |

## 存款

`POST /openapi/v1/mlp/account/deposit` — 权限 `TRADE`。

### 请求体 — MlpDepositReq

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `trackingId` | string | 是 | 幂等追踪 ID |
| `symbolId` | string | 是 | 币种 ID(`ac_symbol`) |
| `amount` | number | 是 | 存款金额,最多 8 位小数 |

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-mlp-dep-0001", "symbolId": "1001", "amount": 100 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/mlp/account/deposit"
```

成功时 `data` 为交易/工作流 ID 字符串。

## 提现

`POST /openapi/v1/mlp/account/withdraw` — 权限 `TRADE`。

从 MLP 转出到资金账户,再划入交易账户。优先扣减本金,再扣减收益。

### 请求体 — MlpWithdrawReq

字段与存款相同:`trackingId`、`symbolId`、`amount`。

```bash cURL
curl -s -X POST \
  -H "Content-Type: application/json" \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  -d '{ "trackingId": "track-mlp-wd-0001", "symbolId": "1001", "amount": 50 }' \
  "https://test-gateway.mcconnects.com/openapi/v1/mlp/account/withdraw"
```

## 图表数据

以下接口接受可选查询参数 `from`、`to`(日期,`yyyy-MM-dd`):

| 路径 | 说明 |
| --- | --- |
| `/openapi/v1/mlp/yesterday-yield-rate-data` | 昨日回报率序列(展示类型为百分比) |
| `/openapi/v1/mlp/tvl-data` | TVL 序列(金额) |
| `/openapi/v1/mlp/apr-data` | APR 序列(百分比) |

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/mlp/tvl-data?from=2026-01-01&to=2026-01-31"
```

## MLP 参数

`GET /openapi/v1/mlp/config` — 权限 `READ`。

返回当前请求上下文业务线下的 MLP 参数,例如 `annualYieldRate`、`lockPeriodDays`、`singleUserDepositLimit`、`tvlBaseAmount`、`integralTierConfig`。

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/mlp/config"
```
