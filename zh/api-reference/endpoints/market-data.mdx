标签 **Market Data（行情数据）** 来自行情聚合服务,用于读取 K 线与排行榜。本组接口均需 `READ` 权限。

## 接口列表

| 方法 | 路径 | 权限 | 描述 |
| --- | --- | --- | --- |
| GET | `/openapi/v1/market-data/kline` | `READ` | 按品种、周期与时间范围查询 K 线。 |
| GET | `/openapi/v1/market-data/symbol-ranking` | `READ` | 按分类与排序维度分页查询行情排行。 |

---

## 查询 K 线

`GET /openapi/v1/market-data/kline` — 权限 `READ`。

### 查询参数 — CompactKlineQueryReq

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `symbol` | string | 是 | 交易品种代码,如 `EURUSD` |
| `period` | int | 是 | 周期(分钟):`1`、`5`、`15`、`60`、`240`、`1440`、`10080`、`43200` 等 |
| `endTime` | long | 是 | 结束时间,**秒级**时间戳 |
| `startTime` | long | 否 | 开始时间,**秒级**时间戳 |
| `limit` | int | 否 | 单次最大返回条数,最大 `500` |

### 响应 `data` — CompactKlineQueryRsp

- `pricingDegree` — 价格精度
- `klineDataList` — 字符串列表,依次表示开、收、高、低、量、秒级时间戳
- `klineDisplayList` — 与列表顺序一致的展示字段

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/market-data/kline?symbol=EURUSD&period=60&endTime=1755878400&limit=100"
```

## 币种行情排行

`GET /openapi/v1/market-data/symbol-ranking` — 权限 `READ`。

### 查询参数 — SymbolRankingQueryReq

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `category` | string | 是 | `CONTRACT` / `SPOT` / `CRYPTO` / `TRADFI` |
| `subGroup` | string | 否 | 仅 `TRADFI`:`ALL` / `FOREX` / `METALS` / `INDICES` / `ENERGY` / `STOCKS`,默认 `ALL` |
| `hotOnly` | boolean | 否 | 仅热门品种,默认 `false` |
| `sortBy` | string | 否 | `symbol` / `bid` / `change24hPct`,默认 `change24hPct` |
| `sortOrder` | string | 否 | `asc` / `desc`(未传时:按 symbol 默认升序,其余降序) |
| `page` | int | 否 | 页码,从 1 开始,默认 `1` |
| `pageSize` | int | 否 | 每页条数,默认 `20`,最大 `100` |

### 响应 `data` — SymbolRankingPageRsp

含 `total`、`page`、`pageSize`、`snapshotVersion`、`items`。

```bash cURL
curl -s \
  -H "X-Access-Key: ${ACCESS_KEY}" \
  -H "X-Signature: ${SIGNATURE}" \
  -H "X-Timestamp: ${TIMESTAMP}" \
  "https://test-gateway.mcconnects.com/openapi/v1/market-data/symbol-ranking?category=CRYPTO&sortBy=change24hPct&sortOrder=desc&page=1&pageSize=20"
```
