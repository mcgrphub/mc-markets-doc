本参考涵盖 MC OpenAPI 的公开接口（仅含标注 `@OpenApiPermission` 的网关路径）。侧边栏 **Account**、**Trade**、**Risk**、**Aggregator** 下的页面由 `openapi/` 中的 Spec 自动生成。

所有接口共享相同的网关主机、鉴权方式（AK/SK + HMAC-SHA256）以及 JSON 响应信封。Spec 中的路径已包含签名所需的服务段（例如 `/openapi/v1/mc-account/...`）。

## 接口分组

| 分组 | Spec | 网关路径前缀 |
| --- | --- | --- |
| Account | `openapi/mc-account.json` | `/openapi/v1/mc-account/` |
| Trade | `openapi/mc-trade.json` | `/openapi/v1/mc-trade/` |
| Risk | `openapi/mc-risk.json` | `/openapi/v1/mc-risk/` |
| Aggregator | `openapi/mc-aggregator.json` | `/openapi/v1/mc-aggregator/` |

在侧边栏打开具体 operation。交互式 Playground 已关闭（请求需自定义 AK/SK 签名），请按 [认证](/zh/authentication) 与 [第一次请求](/zh/guides/first-request) 调用。
