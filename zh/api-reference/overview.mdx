本参考涵盖 MC OpenAPI 的公开接口。侧边栏按 **行情**、**交易**、**理财** 分组。

所有接口共享相同的网关主机、鉴权方式（AK/SK + HMAC-SHA256）以及 JSON 响应信封。请求路径仍包含服务段（例如 `/openapi/v1/mc-account/...`），签名时须使用完整路径。

## 接口分组

| 分组 | 包含 |
| --- | --- |
| 行情 | 行情数据、币种、汇率 |
| 交易 | 下单与持仓、交易账户、订单、成交、账户交易记录、成交订单、资金费历史、交易品种、资金费率 |
| 理财 | 理财账户、金库账户、资金账户 |

在侧边栏打开具体分组。交互式 Playground 已关闭（请求需自定义 AK/SK 签名），请按 [认证](/zh/authentication) 与 [第一次请求](/zh/guides/first-request) 调用。
