**行情数据**接口用于拉取币种行情排行榜与 K 线。本组权限均为 `READ`。路径前缀为 `/openapi/v1/mc-aggregator/`。

## 接口列表

| 方法 | 路径 | 权限 | 描述 |
| --- | --- | --- | --- |
| GET | /openapi/v1/mc-aggregator/market-data/symbol-ranking | READ | 按分类与排序维度分页查询币种行情排行。 |
| GET | /openapi/v1/mc-aggregator/market-data/kline | READ | 按品种、周期与时间范围查询 K 线。 |

## 币种行情排行

`GET /openapi/v1/mc-aggregator/market-data/symbol-ranking` — 权限 `READ`。

按分类与排序维度分页查询币种行情排行榜。

### 查询参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| category | string | 是 | Tab 分类：CONTRACT / SPOT / CRYPTO / TRADFI |
| subGroup | string | 否 | 传统金融二级 Tab：ALL / FOREX / METALS / INDICES / ENERGY / STOCKS |
| hotOnly | boolean | 否 | 是否仅返回热门品种（isHot=1 且 status=1） |
| sortBy | string | 否 | 排序字段：symbol / bid / change24hPct |
| sortOrder | string | 否 | 排序方向：asc / desc |
| page | integer | 否 | 页码，从 1 开始 |
| pageSize | integer | 否 | 每页条数 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| total | integer | 当前分类下品种总数 |
| page | integer | 当前页码 |
| pageSize | integer | 每页条数 |
| snapshotVersion | integer | 排行快照版本号 |
| items | array | 当前页排行条目 |

`items[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| rank | integer | 全局名次 |
| symbol | string | 品种代码 |
| bid | string | 买价 |
| change24hAbs | string | 24h 涨跌额 |
| change24hPct | string | 24h 涨跌幅（小数形式，如 -0.0146 表示 -1.46%） |
| isHot | boolean | 是否热门 |
| pricingDegree | integer | 价格小数位 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-aggregator/market-data/symbol-ranking" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  --data-urlencode "category=SPOT" \
  --data-urlencode "sortBy=change24hPct" \
  --data-urlencode "sortOrder=desc" \
  --data-urlencode "page=1" \
  --data-urlencode "pageSize=20"
```

## 查询 K 线数据

`GET /openapi/v1/mc-aggregator/market-data/kline` — 权限 `READ`。

根据币种、时间间隔与时间范围查询 K 线数据。

### 查询参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| symbol | string | 是 | 交易品种代码（例如 EURUSD） |
| period | integer | 是 | 周期（分钟）：1 / 5 / 15 / 60 / 240 / 1440；周=1440*7；月=1440*30 |
| endTime | integer | 是 | 结束时间，秒级时间戳 |
| startTime | integer | 否 | 开始时间，秒级时间戳 |
| limit | integer | 否 | 单次最大返回条数，最大 500 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| pricingDegree | integer | 价格精度，用于把数值转为浮点数 |
| klineDataList | array | K 线紧凑列表；每项依次为开盘价、收盘价、最高价、最低价、交易量、秒级时间戳 |
| klineDisplayList | array | 与 klineDataList 顺序一致的展示字段 |

`klineDisplayList[]` 常用字段：`openStr` / `openDisplay` / `closeStr` / `closeDisplay` / `highStr` / `highDisplay` / `lowStr` / `lowDisplay` / `volumeStr` / `volumeDisplay` 及对应 `*Scale`，以及 `timestamp`（秒）。

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-aggregator/market-data/kline" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  --data-urlencode "symbol=EURUSD" \
  --data-urlencode "period=60" \
  --data-urlencode "startTime=1755792000" \
  --data-urlencode "endTime=1755878400" \
  --data-urlencode "limit=500"
```
