MC OpenAPI 仅通过 HTTPS 由 **MC API 网关** 提供。每个请求均需使用下方 Base URL \+ 路径前缀(`/openapi/v1`)\+ 具体路径,例如 `https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/usd`。

## Base URL

| | Base URL |
| --- | --- |
| 生产 | `https://gateway.mcconnects.com` |

<Note>
  鉴权方式为 AK/SK \+ HMAC-SHA256。密钥在 MC 平台申请。
</Note>

## 路径前缀

所有资源路径均位于 `/openapi/v1/…` 下。将网关主机与接口路径拼接:

```text
{gateway}/openapi/v1/{resource}/…
```

例如:

- `https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/usd`
- `https://gateway.mcconnects.com/openapi/v1/mc-account/finance/account/overview`

## 示例

<CodeGroup>

```bash cURL
curl -X POST https://gateway.mcconnects.com/openapi/v1/mc-account/exchange-rate/query/usd \
  -H "X-Access-Key: $MC_ACCESS_KEY" \
  -H "X-Signature: $MC_SIGNATURE" \
  -H "X-Timestamp: $MC_TIMESTAMP" \
  -H "Content-Type: application/json"
```

```python Python
import os

MC_GATEWAY = os.environ.get("MC_GATEWAY", "https://gateway.mcconnects.com")
url = f"{MC_GATEWAY}/openapi/v1/mc-account/exchange-rate/query/usd"
```

```javascript Node.js
const MC_GATEWAY = process.env.MC_GATEWAY ?? "https://gateway.mcconnects.com";
const url = `${MC_GATEWAY}/openapi/v1/mc-account/exchange-rate/query/usd`;
```

</CodeGroup>

<Warning>
  始终使用 HTTPS。HTTP 请求会被拒绝。如果客户端自动重定向,POST body 可能不会重发,导致签名校验失败。
</Warning>

<Tip>
  将网关主机存为环境变量(如 `MC_GATEWAY`),便于在应用配置中统一管理。
</Tip>
