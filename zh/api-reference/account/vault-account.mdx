## 金库账户提现

`POST /openapi/v1/mc-account/mlp/account/withdraw` — 权限 `TRADE`。

金库账户提现，从金库账户转出到资金账户，然后划入交易账户。优先扣减本金，再扣减收益。

### 请求体（JSON）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| trackingId | string | 是 | 用于跟踪请求的唯一标识 |
| symbolId | string | 是 | 关联ac_symbol表 |
| amount | number | 是 | 提现金额 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| data | string | `data` 为该类型标量 |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/mlp/account/withdraw" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```

## 查询金库账户参数

查询当前业务线下的金库账户收益率等参数。

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | integer | 配置ID |
| businessLineId | integer | 业务线ID |
| annualYieldRate | number | 调整收益率（年化），单位：百分比 |
| lockPeriodDays | integer | 锁仓期，单位：天 |
| singleUserDepositLimit | number | 单用户存款上限，单位：USDC |
| tvlBaseAmount | number | TVL金额基准值，单位：USDC |
| integralTierConfig | string | 积分档位配置，格式：积分最小值:存款限额比例,...，例如：50:25,300:50,1500:75,10000:100 |
| createBy | string | 创建人 |
| createTime | string | 创建时间 |
| updateBy | string | 更新人 |
| updateTime | string | 更新时间 |
| createTimeTimestamp | integer | 创建时间毫秒时间戳 |
| updateTimeTimestamp | integer | 更新时间毫秒时间戳 |
