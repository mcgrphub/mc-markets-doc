## 资金费历史记录分页查询接口V2

`GET /openapi/v1/mc-account/funding-fee-record/page/v2` — 权限 `READ`。

feeType固定为INTEREST_FEE/FUNDING_FEE，按positionId+dealTime分组汇总amount

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| size | integer | 否 |  |
| current | integer | 否 |  |
| orderBy | string | 否 | 支持多字段排序，用逗号分隔 |
| orderDirection | string | 否 | ASC升序/DESC降序，多字段时用逗号分隔 |
| userId | string | 否 | 用户ID |
| login | string | 否 | mt5登录账号 |
| dealTimeStart | string | 否 | 开始扣费时间 |
| dealTimeEnd | string | 否 | 结束扣费时间 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| records | array |  |
| records[].relatedBizId | string | 关联业务ID |
| records[].userId | string | 用户ID |
| records[].login | string | mt5登录账号 |
| records[].businessLineId | integer | 业务线ID |
| records[].businessLineName | string | 业务线名称 |
| records[].dealTime | string | 扣费时间 |
| records[].symbol | string | 结算币种 |
| records[].symbolPair | string | 交易对 |
| records[].amount | number | 金额汇总，负数表示扣除 |
| records[].remark | string | 备注 |
| records[].createTime | string | 记录创建时间 |
| records[].updateTime | string | 记录更新时间 |
| records[].feeRate | number | 费率 |
| records[].volume | number | 持仓数量 |
| records[].positionSide | string | 持仓方向 |
| records[].positionId | string | 持仓ID |
| records[].batchTime | string | 费用结算批次时间(整点时间) |
| records[].dealTimeTimestamp | integer | 扣费时间毫秒时间戳 |
| records[].feeRateDisplay | string | 资金费率展示值，ratio转百分比后四舍五入至4位小数 |
| records[].feeRateScale | integer | 资金费率展示精度 |
| records[].batchTimeTimestamp | integer | 费用结算批次时间毫秒时间戳 |
| records[].amountDisplay | string | 资金费金额展示值，向上保留8位小数 |
| records[].amountScale | integer | 资金费金额展示精度 |
| records[].createTimeTimestamp | integer | 记录创建时间毫秒时间戳 |
| records[].updateTimeTimestamp | integer | 记录更新时间毫秒时间戳 |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/funding-fee-record/page/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1"
```
