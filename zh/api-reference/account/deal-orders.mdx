## 分页查询成交订单

`GET /openapi/v1/mc-account/deal-order` — 权限 `READ`。

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| size | integer | 否 | 分页每页条数 |
| current | integer | 否 | 分页当前页码，从 1 开始 |
| orderBy | string | 否 | 支持多字段排序，用逗号分隔 |
| orderDirection | string | 否 | ASC升序/DESC降序，多字段时用逗号分隔 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| records | array | 当前页记录列表 |
| total | integer | 总记录数 |
| size | integer | 每页条数 |
| current | integer | 当前页码 |
| pages | integer | 总页数 |
| hasPrevious | boolean | 是否有上一页 |
| hasNext | boolean | 是否有下一页 |

`records[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | integer | 交易记录主键 ID |
| uid | string | 用户 UID |
| transactionId | string | 业务交易流水号 |
| symbolId | integer | 币种 ID |
| acSymbol | string | 币种名称；闪兑时为兑出币种 |
| network | string | 链上网络协议 |
| exchangeSymbolId | integer | 兑换后币种 ID |
| exchangeSymbol | string | 兑换后币种名称；闪兑时为兑入币种 |
| transactionType | string | 交易类型枚举 |
| status | string | 交易状态：SUCCESS / FAIL / PENDING |
| amount | number | 交易金额 |
| amountStr | string | 交易金额字符串 |
| amountDisplay | string | 交易金额展示值 |
| amountScale | integer | 交易金额展示小数位数 |
| exchangeAmount | number | 兑换后到账金额 |
| exchangeAmountDisplay | string | 兑换后金额展示值 |
| fee | number | 交易手续费 |
| feeDisplay | string | 手续费展示值 |
| createTime | string | 交易开始时间 |
| finishTime | string | 交易完成时间 |
| failureReason | string | 失败原因说明 |
| errorCode | string | 失败错误码 |
| exchangeRate | number | 汇率 |
| convertExchangeRate | number | 闪兑汇率（兑入/兑出） |
| rateDisplay | string | 汇率展示值 |
| txHash | string | 链上交易哈希 |
| feeSymbol | string | 手续费计价币种 |
| sourceAccountType | string | 来源账户类型 |
| targetAccountType | string | 目标账户类型 |
| sourceAccountId | string | 来源账户 ID |
| targetAccountId | string | 目标账户 ID |
| sourceWalletAddress | string | 来源链上钱包地址 |
| targetWalletAddress | string | 目标链上钱包地址 |
| extParams | string | 扩展参数 JSON 字符串 |
| sourceAccountBalance | number | 源账户动账后余额 |
| targetAccountBalance | number | 目标账户动账后余额 |
| fundFlowType | string | 资金流向类型 |
| transactionMethod | string | 交易方式枚举 |
| recipientUid | string | WITHDRAWAL 用户间划转时有值 |
| payerUid | string | RECHARGE 用户间划转时有值 |
| finishTimeTimestamp | integer | 交易完成时间毫秒时间戳 |
| createTimeTimestamp | integer | 交易开始时间毫秒时间戳 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/deal-order" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1"
```

## 成交单详情查询

`GET /openapi/v1/mc-account/deal-order/{dealId}` — 权限 `READ`。

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| dealId | integer | 是 | 成交单 ID |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| businessLineId | integer | 业务线ID |
| uid | string | 用户ID |
| positionId | integer | 持仓单ID |
| orderId | integer | 订单号 |
| dealId | integer | 成交单号 |
| dealTime | string | 成交时间 |
| dealTimeTimestamp | integer | 成交时间毫秒时间戳 |
| type | string | 交易类型 |
| symbolPair | string | 交易对 |
| baseSymbolId | integer | 基础币种ID |
| quoteSymbolId | integer | 计价币种ID |
| entry | string | 方向 |
| price | number | 委托价格 |
| pricePosition | number | 成交均价 |
| lot | number | 委托数量 |
| volumeClosed | number | 成交数量 |
| storage | number | 库存费 |
| commission | number | 手续费 |
| commissionDisplay | string | 手续费展示值，带符号向上取整后固定6位小数 |
| commissionScale | integer | 手续费展示精度 |
| profit | number | 交易盈亏额 |
| profitDisplay | string | 盈亏展示值，四舍五入后固定6位小数 |
| profitScale | integer | 盈亏展示精度 |
| contractSize | integer | 合约大小 |
| stopLoss | number | 止损价 |
| takeProfit | number | 止盈价 |
| onChainStatus | string | 上链状态 |
| txHash | string | 区块链交易哈希 |
| retryCount | integer | 重试次数 |
| gasUsed | integer | 实际消耗的Gas |
| gasPrice | integer | Gas价格 |
| blockNumber | integer | 区块号 |
| blockConfirmation | integer | 区块确认数 |
| submitTime | string | 提交时间 |
| submitTimeTimestamp | integer | 提交时间毫秒时间戳 |
| confirmTime | string | 确认时间 |
| confirmTimeTimestamp | integer | 确认时间毫秒时间戳 |
| remark | string | 备注信息 |
| accountType | string | 账户类型 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/deal-order/{dealId}" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "dealId=1"
```
