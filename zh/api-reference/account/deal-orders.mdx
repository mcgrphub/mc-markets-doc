## 分页查询成交订单

`GET /openapi/v1/mc-account/deal-order` — 权限 `READ`。

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| size | integer | 否 |  |
| current | integer | 否 |  |
| orderBy | string | 否 | 支持多字段排序，用逗号分隔 |
| orderDirection | string | 否 | ASC升序/DESC降序，多字段时用逗号分隔 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| records | array |  |
| records[].remark | string | 备注信息 |
| records[].dealAmount | number | 成交价值(USDC) |
| records[].dealAmountDisplay | string | 成交价值展示值 |
| records[].dealAmountScale | integer | 成交价值展示小数位 |
| records[].commission | number | 费用 |
| records[].commissionDisplay | string | 费用展示值 |
| records[].commissionScale | integer | 费用展示小数位 |
| records[].profit | number | 已实现盈亏 |
| records[].profitDisplay | string | 已实现盈亏展示值 |
| records[].profitScale | integer | 已实现盈亏展示小数位 |
| total | integer |  |
| size | integer |  |
| current | integer |  |
| pages | integer |  |
| hasPrevious | boolean |  |
| hasNext | boolean |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/deal-order" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "size=1" \
  --data-urlencode "current=1"
```

| quoteSymbolId | integer | 计价币种ID |
