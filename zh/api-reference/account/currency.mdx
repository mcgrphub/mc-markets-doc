## 查询币种网络列表

`GET /openapi/v1/mc-account/symbol/list` — 权限 `READ`。

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| data | array | `data` 为对象数组，元素字段见 `[]...` |
| [].symbolId | integer | 币种ID |
| [].assetType | string | 币种类型 |
| [].symbol | string | 币种名称 |
| [].network | string | 币种网络协议 |
| [].minTradeUnit | number | 最小动账单位 |
| [].pricePrecision | integer | 价格精度 |
| [].exchangeRate | number | 汇率 |
| [].minRechargeAmount | number |  |
| [].minWithdrawAmount | number |  |
| [].assetNet | string |  |
| [].iconData | string |  |
| [].depositStatus | integer |  |
| [].withdrawalStatus | integer |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/symbol/list" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```

## 查询币种详情

`GET /openapi/v1/mc-account/symbol/detail/{id}` — 权限 `READ`。

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | string | 是 | 币种ID |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| symbolId | integer | 币种ID |
| assetType | string | 币种类型 |
| symbol | string | 币种名称 |
| network | string | 币种网络协议 |
| minTradeUnit | number | 最小动账单位 |
| pricePrecision | integer | 价格精度 |
| exchangeRate | number | 汇率 |
| minRechargeAmount | number |  |
| minWithdrawAmount | number |  |
| assetNet | string |  |
| iconData | string |  |
| depositStatus | integer |  |
| withdrawalStatus | integer |  |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/symbol/detail/{id}" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "id=VALUE"
```
