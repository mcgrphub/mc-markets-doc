## 获取持仓信息V2

`GET /openapi/v1/mc-account/fund/account/spot/positions/v2` — 权限 `READ`。

获取持仓信息

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| uid | string | 用户ID |
| symbolInfo | object | 持仓信息，key为交易品种，value为持仓详情 |

`symbolInfo` 字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| <key> | object | 持仓信息，key为交易品种，value为持仓详情 |

`symbolInfo.<key>` 映射值字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| contractSize | number | 合约大小 |
| freeMargin | number | 可用 |
| precision | number | 精度 |
| priceCurrent | number | 当前价格 |
| symbol | string | 交易品种 |
| usdValue | number | 美元价值 |
| volumeExt | number | 扩展持仓量 |
| volume | number | 实际持仓量 |
| openPrice | number | 开仓价格 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/fund/account/spot/positions/v2" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```

## 获取现货账户持仓信息

`GET /openapi/v1/mc-account/fund/account/spot-account/positions` — 权限 `READ`。

获取现货账户持仓信息

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| spotId | string | 是 | 现货账户 ID |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| spotId | string | 现货账户ID |
| symbolInfo | object | 持仓信息，key为交易品种，value为持仓详情 |

`symbolInfo` 字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| <key> | object | 持仓信息，key为交易品种，value为持仓详情 |

`symbolInfo.<key>` 映射值字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| contractSize | number | 合约大小 |
| freeMargin | number | 可用 |
| precision | number | 精度 |
| priceCurrent | number | 当前价格 |
| symbol | string | 交易品种 |
| usdValue | number | 美元价值 |
| rateProfit | number | 盈亏汇率 |
| volume | number | 实际持仓量 |
| volumeExt | number | 实际持仓量 |
| openPrice | number | 当前价格 |
| freeAmount | number | 可用(不包含挂单占用部分)，只有USDC品种该字段才不为0 |
| volumeExtDec | number | 实际持仓量（volumeExt 按精度换算后的可读值） |
| volumeDisplay | string | 总余额展示值 |
| volumeScale | integer | 总余额展示精度 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/fund/account/spot-account/positions" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
  --data-urlencode "spotId=VALUE"
```

## 查询用户储备金证明（PoR）

`GET /openapi/v1/mc-account/fund/account/por` — 权限 `READ`。

查询当前用户储备金证明（Proof of Reserves）；可按批次号筛选。

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| batchId | integer | 否 | 快照批次号；不传则返回默认/最新可用批次 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| treeBatchId | integer | Merkle Tree构建批次 |
| userId | string | 用户ID |
| userHash | string | PoRv2 用户哈希（user_hash） |
| assetInfo | string | 资产信息 |
| leafHash | string | Merkle leaf 的 hash 值 |
| pathLayers | string | 递归层路径，每层一个 group + 自己在其中的位置 |
| platformAssetProof | array | 平台资产证明 |

`platformAssetProof[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| chain | string | 链名 |
| address | string | 钱包地址 |
| assetInfo | string | 资产信息 |
| signature | string | 签名 |

```bash
curl -sS -G "https://gateway.mcconnects.com/openapi/v1/mc-account/fund/account/por" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE"
```
