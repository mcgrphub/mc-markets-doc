## 理财账户提现（活期）

`POST /openapi/v1/mc-account/finance/account/withdraw` — 权限 `TRADE`。

仅支持活期产品：传入产品代码与提现金额，从理财转出到资金账户再划入现货。定期多笔持仓请使用「定期持仓提现」接口并指定持仓明细 ID。

### 请求体（JSON）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| trackingId | string | 是 | 用于跟踪请求的唯一标识 |
| productCode | string | 是 | 活期理财产品代码 |
| amount | number | 是 | 提现金额 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| records | array | 当前页记录列表 |
| total | integer | 总记录数 |
| size | integer | 每页条数 |
| current | integer | 当前页码 |
| pages | integer | 总页数 |
| hasPrevious | boolean | 是否有上一页 |
| hasNext | boolean | 是否有下一页 |

`records[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | integer | 交易记录ID |
| uid | string | 关联账户表 |
| transactionId | string | 交易ID |
| symbolId | integer | 币种ID |
| acSymbol | string | 币种名/兑出 |
| network | string | 网络协议 |
| exchangeSymbolId | integer | 兑换后币种ID |
| exchangeSymbol | string | 兑换后币种名/兑入 |
| transactionType | string | 交易类型枚举 |
| status | string | 交易状态 |
| amount | number | 交易金额 |
| amountStr | string | 交易金额字符串 |
| amountDisplay | string | 交易金额展示值 |

| exchangeAmount | number | 兑换后金额 |

| fee | number | 交易手续费 |

| targetAccountBalance | number | 目标账户动账后余额 |
| fundFlowType | string | 资金流向类型 |
| transactionMethod | string | 交易方式枚举 |
| recipientUid | string | WITHDRAWAL 用户间划转时有值 |

## 理财概览

`GET /openapi/v1/mc-account/finance/account/overview` — 权限 `READ`。

按产品维度返回 products，顺序为 product_code 升序、再 currency 升序；用户侧 userDepositedAmount/userInterestAmount/availableBalance 来自该产品在 ac_user_finance_position 的持仓汇总（该产品币种）。totalDepositValue 为各产品持仓本金×预言机价折 USDC 之和。totalInterestValue、yesterdayInterestValue 仍为结息流水按币种折 USDC。userSpotAvailableBalance 为现货币种维度，同币种产品值相同。未登录时用户侧为0
