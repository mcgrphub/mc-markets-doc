## 理财账户提现（活期）

`POST /openapi/v1/mc-account/finance/account/withdraw` — 权限 `TRADE`。

仅支持活期产品：传入产品代码与提现金额，从理财转出到资金账户再划入现货。定期多笔持仓请使用「定期持仓提现」接口并指定持仓明细 ID。

### 请求体（JSON）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| trackingId | string | 是 | 用于跟踪请求的唯一标识 |
| productCode | string | 是 | 活期理财产品代码 |
| amount | number | 是 | 提现金额 |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| data | string | `data` 为该类型标量 |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/finance/account/withdraw" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
}' 
```

## 理财定期持仓提现

`POST /openapi/v1/mc-account/finance/account/withdraw/fixed` — 权限 `TRADE`。

仅支持定期产品：指定持仓明细 ID（持有中或已到期均可；与当前持仓接口 `positionId` 一致）。后端按该笔本金与利息全额赎回，工作流与活期提现一致。

### 请求体（JSON）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| trackingId | string | 是 | 用于跟踪请求的唯一标识 |
| positionId | integer | 是 | 定期持仓主键 id（持有中或已到期均可赎回；与当前持仓接口返回的 positionId 一致） |

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| data | string | `data` 为该类型标量 |

```bash
curl -sS -X POST "https://gateway.mcconnects.com/openapi/v1/mc-account/finance/account/withdraw/fixed" \
  -H "X-Access-Key: YOUR_ACCESS_KEY" \
  -H "X-Signature: YOUR_SIGNATURE" \
  -H "X-Timestamp: YOUR_TIMESTAMP" \
  -H "X-Nonce: YOUR_NONCE" \
  -H "Content-Type: application/json" \
  -d '{
  "example": "see fields above"
```

## 理财产品目录

`GET /openapi/v1/mc-account/finance/products/catalog` — 权限 `READ`。

按类型（活期/定期）查询已启用产品配置全量列表（不分页），按基础年化+超额奖励年化降序；查询参数仅 type 必填

### 查询 / 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| type | string | 是 | 产品类型：活期 / 定期 |

### 响应 `data` 字段

`data` 为对象数组。元素字段见下表。

`data[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| productCode | string | 产品代码 |
| productName | string | 产品名称 |
| type | string | 类型：活期/定期 |
| currency | string | 币种 |
| annualRate | number | 基础年化收益率% |
| rewardAnnualRate | number | 超额奖励年化收益率%（配置档） |
| maxAnnualRate | number | 最高年化%=基础+奖励 |
| periodDays | integer | 定期天数，活期为0 |
| minDepositAmount | number | 最低起存金额 |
| minRedeemAmount | number | 最小赎回金额 |
| digit | integer | 币种精度（小数位） |
| subscribedAmount | number | 已申购金额：池内净留存申购本金（与资金池上限校验口径一致：累计存入-累计赎回+累计赎息） |
| fundPoolAmount | number | 池上限：最大资金池；null 或 ≤0 表示不限制 |
| availableSubscriptionBalance | number | 可申购余额：池上限剩余可存入额度；池未设上限时为 null |
| subscriptionStartTime | string | 申购开始时间；与结束时间均为 null 表示不限制 |
| subscriptionEndTime | string | 申购结束时间；与开始时间均为 null 表示不限制 |
| maxDepositAmount | number | 最大申购额：单户最高存入额；null 或 ≤0 表示不限制 |
| dailyMaxRedeemAmount | number | 每日提取上限 |
| subscriptionStartTimeTimestamp | integer | 申购开始时间毫秒时间戳 |
| subscriptionEndTimeTimestamp | integer | 申购结束时间毫秒时间戳 |

## 理财概览

`GET /openapi/v1/mc-account/finance/account/overview` — 权限 `READ`。

按产品维度返回 products，顺序为 product_code 升序、再 currency 升序；用户侧 userDepositedAmount/userInterestAmount/availableBalance 来自该产品在 ac_user_finance_position 的持仓汇总（该产品币种）。totalDepositValue 为各产品持仓本金×预言机价折 USDC 之和。totalInterestValue、yesterdayInterestValue 仍为结息流水按币种折 USDC。userSpotAvailableBalance 为现货币种维度，同币种产品值相同。未登录时用户侧为0

### 响应 `data` 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| totalDepositValue | number | 用户存入总额（各币种本金按参考价折 USDC；未登录为 0） |
| totalInterestValue | number | 用户历史结息总额（按币种汇总后折 USDC；未登录为 0） |
| yesterdayInterestValue | number | 昨日利息（折 USDC；未登录为 0） |
| products | array | 按产品维度的明细列表（含用户持仓汇总与产品申购/资金池字段，口径与理财产品目录一致） |
| totalDepositValueDisplay | string | 用户存入总额展示值 |
| totalInterestValueDisplay | string | 用户历史结息总额展示值 |
| yesterdayInterestValueDisplay | string | 昨日利息展示值 |

`products[]` 元素：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| productCode | string | 产品代码 |
| type | string | 产品类型：活期 / 定期 |
| currency | string | 币种 |
| lockPeriodDays | integer | 锁仓天数（定期）；活期为 0 |
| maxDepositAmount | number | 单用户申购上限（单户最高存入额；null 或 ≤0 表示不限制） |
| dailyMaxRedeemAmount | number | 每日赎回上限（null 或 ≤0 表示不限制） |
| pauseRedeem | boolean | 是否暂停赎回（true 表示已暂停，不可发起赎回） |
| annualRate | number | 基础年化收益率%（不含积分奖励） |
| rewardAnnualRate | number | 奖励年利率：超额奖励档年化% |
| totalAnnualRate | number | 总年化%=基础+奖励（展示用） |
| minDepositAmount | number | 最小存入金额 |
| minRedeemAmount | number | 最小提取金额 |
| subscriptionStartTime | string | 申购开始时间；与结束时间均为 null 表示不限制 |
| subscriptionEndTime | string | 申购结束时间；与开始时间均为 null 表示不限制 |
| rewardMinHoldingDays | integer | 订单持有满天数：享受超额奖励档需连续持仓满（天） |
| rewardMinPosition | number | 订单持有金额不低于：与持仓天数同时满足才分层计息 |
| oraclePrice | number | 计价参考价格（折算用） |
| userDepositedAmount | number | 当前用户在该产品下的持仓本金合计（持有中+已到期；未登录为 0） |
| userInterestAmount | number | 当前用户在该产品下的未结利息合计（未登录为 0） |

| availableSubscriptionBalance | number | 可申购余额：池上限剩余可存入额度；池未设上限时为 null |
| digit | integer | 币种小数位 |
