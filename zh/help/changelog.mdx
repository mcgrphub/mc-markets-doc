<Update label="2026-07-21" description="API 参考改版">
  - API 参考侧边栏按 **行情** / **交易** / **理财** 分组展示（不再按 Spec 服务 Account / Trade / Risk / Aggregator）。
  - 对外统一使用 **金库账户**；移除已下线的 App 全局搜索接口。
  - Playground 关闭（需自定义 AK/SK 签名）。
</Update>

<Update label="2026-07-20" description="开放接口扩展">
  - **鉴权签名**：更新 [认证](/zh/authentication) 与 [快速开始](/zh/quickstart)。必须携带 `X-Nonce`；`POST`/`PUT`/`PATCH` 非空 body 需纳入 SHA-256；`X-Signature` 为 Base64。同步修正请求/响应、FAQ、排障中的过时签名说明。
  - **理财账户**:新增产品目录、当前持仓、赎回明细;活期/定期赎回权限调整为 `TRADE`。
  - **交易**:新增闪兑执行 `POST /openapi/v1/transaction/exchange/v2`(`TRADE`)。
  - **成交订单**:开放成交分页与详情。
  - **金库账户**:开放存取、资产、图表与参数查询。
  - **资金账户**:新增按 `spotId` 查询现货持仓。
  - **资金费率 / 交易品种**(mc-risk):开放资金费率历史分页与交易品种列表。
  - **行情数据**(mc-aggregator-service):开放 K 线与行情排行。
</Update>

<Update label="2026-07-16" description="文档刷新">
  - 发布 MC OpenAPI 的完整文档站点,覆盖六个接口分组:交易、理财账户、币种、资金账户、资金费、汇率。
  - 新增 [认证](/zh/authentication) 参考,包含 HMAC-SHA256 待签名字符串规范与 Python、Java、Node.js 拷贝即用示例。
  - 新增 [频率限制](/zh/guides/rate-limits),含指数退避封装。
  - 新增 [轮询更新](/zh/guides/webhooks) — MC OpenAPI 仅支持请求-响应,使用轮询模式检测新活动。
  - 新增 Python、Node.js、Java、Go [开箱即用客户端](/zh/guides/sdks)。
</Update>

<Update label="v2 接口" description="推荐使用">
  以下 v2 接口取代对应 v1。v1 仍可用,可自行安排迁移。

  - `GET /openapi/v1/mc-account/fund/account/spot/positions/v2` — 现货持仓,扩展字段。
  - `POST /openapi/v1/mc-account/funding-fee-record/page/v2` — 按 `positionId + dealTime` 聚合的资金费历史。

  完整见 [版本管理](/zh/api-reference/versioning)。
</Update>

<Update label="初始 API 范围" description="MC OpenAPI v0.0.1">
  MC OpenAPI 正式上线,包含以下分组:

  - **交易** — 按 ID 或分页查询交易记录。
  - **理财账户** — 活期/定期产品申购/赎回与投资组合概览。
  - **币种** — 币种网络、可兑换币种与币种详情。
  - **资金账户** — 现货持仓与储备金证明(PoRv2)快照。
  - **资金费** — 永续合约资金费历史。
  - **汇率** — 币对、USD、闪兑汇率查询。

  所有接口共享路径前缀 `/openapi/v1/`,使用 AK/SK \+ HMAC-SHA256 签名。
</Update>
