## Get started in 3 steps

<Steps>
  <Step title="Get your API key">
    Sign in to the [dashboard](https://dashboard.example.com) and generate an API key from **Settings → API Keys**.
  </Step>
  <Step title="Make your first request">
    Pass your key in the `Authorization` header and call any endpoint:
    ```bash
    curl https://api.example.com/v1/users \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Step>
  <Step title="Explore & build">
    Browse the [API Reference](/api-reference/overview) for every available endpoint, or follow the [Guides](/guides/first-request) to tackle common workflows.
  </Step>
</Steps>
