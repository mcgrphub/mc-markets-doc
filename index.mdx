Use it to query trading and fund-flow records, manage finance-account deposits and withdrawals, look up symbols and networks, retrieve exchange rates and flash-conversion quotes, and read positions and proof-of-reserves data.

All requests are routed through the MC **API Gateway** and authenticated with an **Access Key / Secret Key** pair using **HMAC-SHA256** request signing.

## Get started in 3 steps

<Steps>
  <Step title="Get your API key">
    Log in to the MC platform, open **API Key Management**, create a new key, and select the required permissions (`READ`, `TRADE`, `WITHDRAW`). Copy the Access Key and Secret Key — the Secret Key is shown only once.
  </Step>
  <Step title="Sign your request">
    Build the string to sign (including `X-Nonce`, and body SHA-256 when required), compute an HMAC-SHA256 signature with your Secret Key (**Base64**), and send it with your Access Key, millisecond timestamp, and nonce. See [Authentication](/authentication) for signature v2.
  </Step>
  <Step title="Call an endpoint">
    Try the [exchange-rate query](/api-reference/endpoints/exchange-rate) to verify your integration, then browse the [full API reference](/api-reference/overview).
  </Step>
</Steps>
