This quickstart walks you through the steps to make your first authenticated call to the MC OpenAPI: creating an API Key, signing a request with HMAC-SHA256, and calling the exchange-rate endpoint through the gateway.

<Note>
  You need an MC platform account before you begin. If you don't have one, contact your MC representative.
</Note>

<Steps>
  <Step title="Create an API Key">
    Log in to the MC platform, open **API Key Management**, and click **Create**. Give the key a descriptive name and select the permissions you need:

    - `READ` — read-only operations (balances, lists, queries)
    - `TRADE` — trading operations (place / cancel orders)
    - `WITHDRAW` — withdrawal and transfer operations

    Optionally configure an IP whitelist (up to 15 IPv4 addresses). Copy the **Access Key** and **Secret Key** immediately — the Secret Key is shown only once.

    <Warning>
      Never commit the Secret Key to source control, and never expose it in client-side code.
    </Warning>
  </Step>
  <Step title="Sign a request">
    Every authenticated request needs three headers:

    | Header | Value |
    | --- | --- |
    | `X-Access-Key` | Your Access Key |
    | `X-Timestamp` | Current Unix time in **milliseconds** (must be within ±10 minutes of server time) |
    | `X-Signature` | Base64-encoded HMAC-SHA256 of the string to sign |

    Build the string to sign by joining the following parts with `\n`, then append a trailing `\n`:

    ```text
    {HTTP_METHOD}
    {REQUEST_URI}
    {QUERY_STRING}       # omit this line if there are no query parameters
    {SIGNED_HEADERS}     # x-access-key={AK}&x-timestamp={TS}, keys lowercased & sorted
    {TIMESTAMP}
    ```

    See [Authentication](/authentication) for the exact rules and Python / Java / Node.js code samples.
  </Step>
  <Step title="Call the exchange-rate endpoint">
    The `GET /openapi/v1/mc-account/exchange-rate/query/CRYPTO/BTC/USDT` endpoint requires only the `READ` permission and is ideal for verifying your signing implementation.

    ```bash
    ACCESS_KEY='YOUR_ACCESS_KEY'
    SECRET_KEY='YOUR_SECRET_KEY'
    URI='/openapi/v1/mc-account/exchange-rate/query/CRYPTO/BTC/USDT'
    TIMESTAMP=$(python3 -c 'import time; print(int(time.time()*1000))')
    
    SIGNATURE=$(python3 << 'PY'
    import base64, hashlib, hmac, os
    method = "GET"
    uri = os.environ["URI"]
    access_key = os.environ["ACCESS_KEY"]
    secret_key = os.environ["SECRET_KEY"]
    timestamp = os.environ["TIMESTAMP"]
    headers = f"x-access-key={access_key}&x-timestamp={timestamp}"
    string_to_sign = f"{method}\n{uri}\n{headers}\n{timestamp}\n"
    sig = base64.b64encode(
        hmac.new(secret_key.encode(), string_to_sign.encode(), hashlib.sha256).digest()
    ).decode()
    print(sig)
    PY
    )
    
    curl -s \
      -H "X-Access-Key: ${ACCESS_KEY}" \
      -H "X-Signature: ${SIGNATURE}" \
      -H "X-Timestamp: ${TIMESTAMP}" \
      "https://test-gateway.mcconnects.com${URI}"
    ```

    A successful response:

    ```json
    {
      "status": 200,
      "code": "200",
      "success": true,
      "data": 98500.12
    }
    ```
  </Step>
